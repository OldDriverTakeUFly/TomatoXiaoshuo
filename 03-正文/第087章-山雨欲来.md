# 第八十七章 山雨欲来

沈暮雪突破 B 级后的第三天。

裴夜带回了一个坏消息。

"南域——派了使者来。"

"又来？方明远？"

"不是方明远。"裴夜说，"是南域的二把手——副指挥官，秦岳。B 级力量强化型。"

"他来干什么？"

"名义上是'商讨两基地合作事宜'。"裴夜说，"实际上是——最后通牒。"

沈暮雪的眉头紧锁。

"最后通牒？"

"南域给青北两个选择。"裴夜说，"第一，接受南域的'保护'——交出军事指挥权，青北成为南域的附属基地。"

"第二——"

"开战。"

沈暮雪站在那里，沉默了。

这就是末世里的"外交"。

没有谈判，没有折中——

只有"服从"或"战争"。

"韩指挥怎么回应的？"

"还没回应。"裴夜说，"秦岳给了七天期限。七天后——如果不答复——南域视为青北拒绝。"

"到时候——"

"两百个异能者，五个 B 级——会兵临城下。"

七天。

沈暮雪深吸一口气。

七天时间——她需要做好一切准备。

"裴夜。"

"嗯。"

"南域——真的会打吗？"

"会。"裴夜说，"南域的头领——前世叫什么来着——我不记得了。但他的性格我记得——"

"他是个扩张主义者。末世里，他认为只有统一所有基地，才能集中力量对抗变异兽潮。"

"青北是他必须拿下的——因为青北的地理位置和防御工事，加上裴夜这个 S 级的存在——"

"如果不拿下青北——青北就是他背后的隐患。"

沈暮雪明白了。

这不是"要不要打"的问题——

是"必须打"。

南域不会让一个有 S 级强者的基地存在于自己的侧翼。

要么收服——要么消灭。

没有第三条路。

"那——我们的选择就只有一个了。"

"打。"裴夜说。

"打。"沈暮雪重复了一遍。

---

当天下午。

沈暮雪召集了所有核心成员——在板房里开战前会议。

参加的人有：裴夜、陈雪、霍北、宋老、姜糖、小周、张姐。

以及——通过视频通讯参加的韩志远和赵德明。

沈暮雪站在地图前，把南域的兵力部署和进攻路线——全部标注了出来。

"南域的三个战斗团——分别从东南、南、西南三个方向进攻。"

"东南方向是主力——由秦岳亲自带队，两个 B 级异能者随行。"

"南方是佯攻——一个 B 级带五十个异能者。"

"西南是迂回——目标是切断青北和北方的联系，封锁我们的退路。"

"总兵力——约两百人。五个 B 级。"

她放下笔，看向众人。

"我们的兵力——"

"我（B 级双系）、裴夜（S 级）、陈雪（B 级）。"

"加上十几个 D 级异能者和三十多个 E 级。"

"总兵力——约六十人。"

兵力对比——一比三。

B 级以上对比——三比八（青北三人，南域五个 B 级加秦岳）。

劣势——很明显。

但——

"我们有三个优势。"沈暮雪说。

"第一——地利。我们守城，他们攻城。城墙和防御工事是我们的天然优势。"

"第二——裴夜。S 级对 B 级是碾压。秦岳是 B 级——在裴夜面前走不了一招。"

"第三——"沈暮雪看了看自己的手，"我。B 级双系。在南域的情报里——我还是 C 级。他们不知道我突破了。"

"情报差——是我们的最大武器。"

韩志远的声音从通讯器里传来："沈小姐——你的意思是？"

"南域的情报来源是陆衍。"沈暮雪说，"陆衍离开的时候——我是 C 级。南域的作战计划是针对'C 级沈暮雪'制定的。"

"但实际上——我是 B 级。"

"当他们遇到'B 级双系'的我——"

"他们的计划会全盘崩溃。"

会议室里安静了几秒。

然后陈雪笑了。

"好一个情报差。"她拍了拍桌子，"南域以为自己来打一个 C 级小丫头——结果迎面撞上一个 B 级双系怪物——"

"够他们喝一壶的。"

沈暮雪也笑了。

"所以——我的方案是。"

她在地图上画了三个圈。

"裴夜——对秦岳。S 级对 B 级——秒杀。尽快结束战斗后支援其他方向。"

"陈雪——守南段。对付南域的佯攻部队。"

"我——去东南段。正面迎击南域主力。"

"霍北——守城墙。用土墙阵阻止敌人靠近。"

"姜糖——医疗站。和宋老一起。"

"其余异能者——配合城防军，守城墙。"

她看向所有人。

"这个方案——有一个关键点。"

"我必须在裴夜解决秦岳之前——拖住南域的主力。"

"一个人——对两个 B 级和一百个异能者。"

会议室里又安静了。

"沈暮雪。"韩志远的声音传来，"你确定——你能拖住？"

沈暮雪抬起右手。

白金色的火焰——在掌心凝聚。

整个会议室的温度——骤升。

桌上的水杯——水开始冒热气。

陈雪的短发——被热浪吹得飘了起来。

所有人都瞪大了眼。

白金色。

那不是 C 级的金红色——

是 B 级的白金色。

"我能。"

沈暮雪收起火焰。

"而且——不只是拖住。"

"我要——赢。"
