# 第八十章 四面楚歌

天亮了。

但太阳没有照常升起——天空被一层灰蒙蒙的、不自然的云层遮住了。

云层很低，像是一块灰色的幕布压在头顶。

空气闷热得让人喘不过气。

沈暮雪站在南段城墙上，看着远方。

她看见了。

四只 C 级变异兽。

它们不再像上次那样躲在兽群后面——

这一次，它们走在最前面。

四只巨兽，从四个方向，向青北基地逼近。

南面——那只老面孔。暗金色皮毛的变异虎。琥珀色的眼睛冷冷地注视着城墙。它的左肩上还有一道旧疤——上次裴夜的雷矛留下的。

东面——一条变异蟒。体长超过二十米，浑身覆盖着墨绿色的鳞片，鳞片表面泛着油亮的光泽。它没有脚——但它移动的速度快得惊人，像一条绿色的河流在地面上流淌。

西面——一只变异鹰。翼展至少五米，浑身的羽毛是铁灰色的，像钢铁一样坚硬。它的喙和爪子闪着寒光——那不是普通的角质，是某种金属化的变异组织。

北面——从地下来的。地面上能看到一道道移动的土丘——那是变异蝎在地下挖掘隧道。偶尔能看到一条带着毒刺的尾巴从土里冒出来，又缩回去。

四只 C 级。

四个方向。

地上、地下、空中——全覆盖。

"这就是C级变异兽的协同战术。"裴夜站在沈暮雪旁边，面色凝重，"它们不是各自为战——是有分工的。"

"虎和蟒正面攻城——吸引守军注意力。"

"鹰从空中突袭后方——打击医疗站和指挥系统。"

"蝎从地下潜入——破坏城墙地基，制造内乱。"

沈暮雪深吸一口气。

"广播。"她说。

韩志远的声音从基地广播里传出来——

"全体基地人员注意。第二轮兽潮——正式开始。"

"四只 C 级变异兽分别从东南西北四个方向进攻。所有战斗人员各就各位。"

"南段——沈暮雪、陈雪。"

"东段——裴夜将军。"

"西段——城防军加 D 级异能者组。"

"北段——霍北负责地下防御。"

"医疗队——全员待命。"

"这一战——不成功，便成仁。"

广播结束。

城墙上，所有人各就各位。

沈暮雪站在南段城墙的最高点。

小豆子趴在她的兜帽里，冰蓝色的眼睛盯着远方——

"嘤嘤嘤！"它急促地叫了几声，连续指了四个方向。

沈暮雪瞬间掌握了四只 C 级变异兽的精确位置。

"陈雪——变异虎在南面正前方八百米，速度匀速，在蓄力。"

"裴夜——变异蟒在东面九百米，移动速度快，预计十五分钟到达城墙。"

"西面的鹰——在空中盘旋，高度约两百米。等地面进攻开始后，它会俯冲。"

"北面的蝎——已经在地下了。深度大约三米，正在向城墙方向挖掘。"

她把情报通过对讲机传给每一段的指挥官。

整个基地的防御系统——因为小豆子的侦察——获得了精确到分钟的预警。

"变异蟒十五分钟后到达东段城墙。"裴夜的声音从对讲机里传来，"我先去解决它。"

"小心。"

"嗯。"

裴夜的身影从南段城墙上消失——S 级的速度，一闪就没了。

沈暮雪转向南面。

变异虎。

上次差点杀了她的那只。

"又见面了。"她低声说。

变异虎在八百米外停下了脚步。

琥珀色的眼睛——和她的眼睛对上了。

它认出了她。

那个上次用火焰旋涡烧伤了它的女人。

那个在它面前灵气耗尽、跪在地上的猎物。

变异虎的嘴角咧开——

露出了獠牙。

上次——它差一点就杀了她。

这一次——

沈暮雪抬起右手。

金红色的火焰在掌心凝聚。

不是暗红色。

是金红色。

C 级的颜色。

变异虎的瞳孔——收缩了一下。

它感觉到了。

这个人类——变强了。

不是上次那个 D 级的猎物了。

但——

变异虎没有退。

它是 C 级。

在它的世界里——

没有退。

只有——

杀。

"嗷————！！！"

它吼了。

冲锋开始了。

四面楚歌的青北基地——

迎来了它最漫长的一天。

而沈暮雪——

站在南段城墙上。

面朝变异虎。

身后是她要守护的一切。

"来吧。"她说。

金红色的火焰在她手中——

燃烧。
