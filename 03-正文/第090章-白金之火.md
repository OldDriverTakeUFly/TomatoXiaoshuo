# 第九十章 白金之火

1500 度。

这个温度意味着什么？

铁的熔点——1538 度。

也就是说——沈暮雪的炎帝领域，几乎可以熔化铁。

南域冲锋在前的异能者们——大部分是 E 级和 D 级——

他们冲进了三十米范围——

然后——

"啊啊啊——！！"

惨叫声。

第一个冲进来的 E 级异能者——他的异能护盾在 1500 度面前像纸一样——

瞬间蒸发。

他整个人——衣服、头发、皮肤——在零点几秒内被灼烧。

他惨叫着往后退——但已经来不及了。

高温——不是火焰——是纯粹的、无处不在的热量。

空气本身在燃烧。

第二个、第三个、第四个——

冲在前面的十几个 E 级异能者——全部被炎帝领域笼罩——

他们的异能、护盾、甚至武器——

在 1500 度面前——

毫无意义。

"退——退出去——！！"

南域的指挥官尖叫着——

冲锋的队伍瞬间崩溃——前面的人拼命后退，后面的人还在往前冲——

混乱。

沈暮雪站在城墙上，精准地控制着领域的范围——

只覆盖城墙外侧三十米——不碰城墙，不碰友军。

她的控制力——在三天前还做不到这个程度。

但现在是战前——五天的苦练——

她做到了。

"这是——B 级？！"

南域的 B 级火系异能者——一个三十多岁的红发男人——在三百米外看见了那片白金色的火海——

他的瞳孔骤缩。

他是 B 级火系——他知道白金色火焰意味着什么。

那是——B 级。

而且——

"她控制得这么精准？！"红发男人震惊了——B 级的炎帝领域极难控制，能在实战中精确到"城墙外侧三十米不伤友军"——

这不是普通的 B 级。

"情报说她是 C 级——"他咬着牙，"陆衍那个废物——给的是什么狗屁情报！"

但——

他是 B 级火系。

同系——他有部分抵消能力。

"我去。"红发男人对旁边的土系 B 级说，"你从侧面绕——用土墙挡住她的火焰——我来正面抵消她的领域。"

"小心——她是双系。"

"我知道。"

红发男人冲了出去——

他冲向了沈暮雪的炎帝领域——

作为 B 级火系——他可以在 1500 度的领域中存活——因为同系的火焰对他伤害减半。

但他进入领域的那一刻——

他感受到了——

沈暮雪的火焰——不只是热。

还有——压制。

炎帝领域的附带效果——"压制范围内所有低于 B 级的异能者灵气运转"。

他是 B 级——没有被压制——

但他感觉到了——沈暮雪的灵气浓度，比他高。

"她的火系——比我强？"红发男人心里一惊。

不可能——他可是南域最强的火系 B 级——

但沈暮雪的灵气——

是灵泉和灵果养出来的、纯度极高的火系灵气。

比普通觉醒者的灵气——浓郁了数倍。

"同系抵消？"沈暮雪看着冲进来的红发男人，冷笑了一下，"你以为——同系就能挡住我？"

她右手一挥——

不是火焰——

是空间。

她从空间里取出了一颗——C 级火系晶核。

直接——握在掌心。

晶核里的灵气——在 B 级火系的催动下——瞬间被激活——

形成了一道——

"火凤——"

沈暮雪双手合十，把晶核的灵气和自己的 B 级灵气融合——

凝聚成一只——

白金色的、展翅的火凤。

"去。"

火凤展翅——

三米翼展的白金色火焰之鸟——

从她的掌心飞出——

直扑红发男人。

"什么——！！"

红发男人本能地举起双手——火系灵气全开——试图抵消——

但火凤不是普通的火焰——

它是沈暮雪的 B 级灵气 + C 级晶核灵气 + 空间系精密控制——三重加持下的产物。

红发男人的抵消——

只撑了三秒。

火凤穿透了他的防御——

"嗷——！！"

他全身被白金色的火焰点燃——

B 级的火系护甲——在火凤面前——

像蜡遇到了火。

他惨叫着——往回跑——

但火凤附在身上——不灭。

他在地上翻滚、用灵气扑打——

火不灭。

沈暮雪的白金色火焰——不是普通的火——

它附着在灵气上——只要对方还有灵气——火就不会灭。

红发男人——

在众目睽睽之下——

被烧成了一团白金色的火球——

倒在了地上。

不再动弹。

B 级火系——被同系的沈暮雪——

一招击杀。

城墙上的南域军队——

彻底崩溃了。

"B 级——她杀了 B 级——"

"一个火凤——就杀了——"

"这不是 B 级——这是怪物——"

"撤退——撤退——！！"

而另一边——

那个土系 B 级——

看见红发男人被杀——

转身就跑。

他连打都没打——直接跑了。

B 级对 B 级——如果红发男人三秒就被杀了——他一个土系——凭什么打？

跑。

南域的主力——

一百人——

在沈暮雪一个人面前——

溃败了。

---

与此同时。

六百米外。

秦岳正等着前方的战报——

等来的却是——溃败的消息。

"什么？！"他抓住传令兵的衣领，"一百人——被一个女人——打溃了？！"

"她——她是 B 级——不是 C 级——白金色的火焰——一招杀了张锋（红发男人）——"

秦岳的脸色——铁青。

B 级。

沈暮雪是 B 级。

陆衍的情报——说她是 C 级。

"陆衍——！！"秦岳转头，目光如刀地射向后方——

陆衍的脸——惨白。

他——

他不知道沈暮雪突破了。

他离开的时候——沈暮雪确实是 C 级。

但那是——十几天前。

这十几天里——她吸收了四颗 C 级晶核——突破到了 B 级。

陆衍——

提供了一份过时的情报。

而南域——

基于这份过时的情报——

制定了一个注定失败的作战计划。

"你——"秦岳走到陆衍面前，一字一句，"你给我的情报——是错的。"

陆衍的腿在抖。

"我——我走的时候她确实是 C 级——我——"

"闭嘴。"

秦岳的拳头——握紧了。

B 级力量强化型的拳头——

如果这一拳打在陆衍身上——

他会变成一团肉泥。

陆衍闭上了眼——

就在这时——

"轰——！！"

一道白紫色的闪电——

从天而降——

劈在了秦岳面前三米处的地面上。

地面被劈出了一个两米深的坑——

焦黑的、冒着青烟的坑。

秦岳——

后退了两步。

他抬头——

一个黑色的身影——从天而降——

落在了他面前。

裴夜。

"秦岳。"裴夜的声音很低，"你的军队——正在溃败。"

"你的两个 B 级——一个死了，一个跑了。"

"现在——轮到你了。"

秦岳看着裴夜——

B 级力量强化型——他的本能告诉他——

眼前这个人——

危险。

极度危险。

但他是 B 级——南域第二强者——

他不信自己连一招都撑不过——

"呵——"

秦岳举起拳头——

全力一拳——

朝裴夜砸了过去——

B 级力量强化型的全力一拳——

足以掀翻一辆装甲车——

裴夜——

抬起一只手。

食指。

一道白紫色的电弧——

从指尖射出——

点在了秦岳的拳头上。

"嗤——！"

秦岳的拳头——在半空中停住了。

不是他主动停的——

是他的手臂——被雷系灵气麻痹了。

整条右臂——从拳头到肩膀——

失去了知觉。

"什——"

秦岳还没来得及说完——

裴夜的第二道闪电——

已经到了。

"轰——！！"

正中胸口。

秦岳的身体——被闪电击飞了五米——

重重地摔在地上——

胸口的作战服被烧穿了一个洞——

下面的皮肤——焦黑。

他躺在地上——

想站起来——

但身体——不听使唤。

S 级的雷系——

对他的 B 级身体——

造成了全身性的神经麻痹。

裴夜走过去——

居高临下地看着他。

"三十秒。"裴夜说，"我说过——三十秒。"

秦岳躺在地上，瞪着他——

眼里满是——

恐惧。

S 级。

这才是 S 级。

他——一个 B 级——

连一招都接不住。

裴夜转身——

看向远处正在溃逃的南域军队——

以及——

站在后方、脸色惨白的——

方明远和陆衍。

裴夜的目光——

在陆衍身上停留了一秒。

然后——

他收回目光。

陆衍——

不是他的人。

是——

沈暮雪的仇人。

该由她——来处理。
