# 第五十三章 寻宝

第一波冲击来了。

三百多只变异兽同时冲锋，像一道黑色的洪流涌向城南围墙。

城墙上的重机枪率先开火——

"哒哒哒哒哒——！"

子弹像雨点一样泼洒下去，打在最前排的 D 级精英犬身上。

但这些变异犬的皮肉已经硬化到了一定程度——普通子弹只能打伤它们，不能一击致命。

它们顶着弹雨冲锋，被打得血肉模糊，但不减速。

"妈的，皮太厚了！"一个机枪手骂道。

异能者也出手了。

风系异能者用风刃切割——嗖嗖嗖的风刃飞入兽群，将几只冲在最前面的变异犬斩成两半。

土系异能者在城墙前方升起了一道土墙——挡住了第一波冲击。

但兽群太多了。

土墙被撞碎，变异犬踩着同伴的尸体继续冲。

"弓箭手！射前排！"陈雪的声音在城墙上响起。

她站在城墙的最前沿，短发被风吹得猎猎作响，左手挥出一道风刃，右手握着一把军刀。

B 级风系——她的风刃比其他风系异能者强了十倍不止，一刀下去能斩三只变异犬。

城墙上的异能者开始用弩箭射击——箭矢带着各自的异能属性飞入兽群。

火系、冰系、雷系（低级的）——各种颜色的光芒在兽群中闪烁。

但即使如此——

兽群的数量优势太大了。

前排的倒下了，后排的立刻补上。

三百只打掉了一半——还有一百五十只。

打掉一百五十只——还有七十五只。

而城墙上的弹药和异能消耗，在飞速下降。

沈暮雪站在补给站后面，密切关注着战场局势。

她的手按在空间上——随时准备取出弹药和箭矢补给前线。

但她知道——光靠补给是不够的。

必须找到更高效的战斗方式。

而那个方式——

"嘤。"

口袋里又响了。

沈暮雪低头。

小豆子又探出了半个脑袋。

它的冰蓝色眼睛盯着城墙外的兽群，鼻尖微微翕动——

然后，它伸出一只小爪子，指向了一个方向。

城墙左前方，约两百米处。

那个位置——在密集的兽群中，并不起眼。

但小豆子的爪子指得很坚定。

"嘤！"

沈暮雪的瞳孔收缩了一下。

她明白了。

小豆子感知到了什么。

那个位置——有一只高等级的变异兽。

它的晶核——被小豆子的寻宝天赋捕捉到了。

沈暮雪拿起望远镜，对准小豆子指的方向——

在密集的普通变异兽之间，她看到了一只——

比周围的变异犬大一圈的、毛色偏灰的个体。

它不像其他变异兽那样疯狂冲锋，而是在兽群中游走，时不时发出低沉的嚎叫——

在指挥。

"D 级精英。"沈暮雪低声说，"它在指挥前排的冲锋。"

如果能杀掉它——兽群的协同攻击就会瓦解，变成各自为战的散兵游勇。

"小豆子，"沈暮雪低声说，"还能找到别的吗？"

小豆子歪了歪头，然后闭上了眼。

它的银白色绒毛微微发光——

几秒钟后，它睁开眼，连续指了三个方向。

一个在城墙正前方——那只灰色的指挥犬。

一个在城墙右前方——一个隐藏在兽群后面的、体型较小的变异兽，可能是负责侦察的。

一个在更远的地方——

沈暮雪顺着它指的方向看去。

五百米外，兽群的后方。

那个位置——

有一个巨大的、蜷缩着的影子。

C 级。

它还没有动。

在等。

沈暮雪心里一凛。

"好孩子。"她摸了摸小豆子的头，"回去。"

小豆子缩回了空间。

沈暮雪拿起对讲机。

"陈队长。"

"收到。"陈雪的声音从对讲机里传来，伴随着风刃呼啸的背景音。

"城墙左前方两百米，灰色的大号变异犬——它是这一波的指挥官。杀了它，兽群会散。"

对面沉默了一秒。

"你怎么知道的？"

"我自己的侦察手段。"沈暮雪说，"信不信由你。"

陈雪沉默了两秒。

然后——

"收到。"

对讲机断了。

沈暮雪透过观察孔看出去——

她看见陈雪在城墙上移动了位置，跑到了左前方。

陈雪举起手，风系灵气在掌心凝聚——

一道比之前大了三倍的风刃，呼啸着飞出——

两百米外，灰色的指挥犬正在嚎叫指挥——

风刃精准地切中了它的脖子。

"嗷——！"

灰犬的嚎叫变成了一声惨叫。

它的头——被风刃切掉了一半。

倒下了。

效果立竿见影。

灰犬一死，前排的变异兽瞬间失去了协同——

有的还在冲锋，有的开始东张西望，有的甚至转头跑了。

兽群的冲锋队形，瓦解了。

城墙上的守军抓住机会——重机枪和异能集中火力，把失去组织的兽群像割麦子一样扫倒。

第一波——

打退了。

城墙上一片欢呼。

陈雪放下手，喘了口气，然后按下对讲机。

"沈暮雪。"

"在。"

"那个情报——准得很。"陈雪的声音里多了一分认真，"下次有类似的情况，第一时间通知我。"

"没问题。"

"还有——"陈雪顿了一下，"你那个'侦察手段'，是什么？"

"秘密。"沈暮雪笑了笑。

对讲机那头沉默了一秒，然后传来陈雪的一声轻笑。

"行。你的秘密，你留着。"

"只要管用就行。"

对讲机断了。

沈暮雪站在补给站后面，嘴角微微弯起来。

第一波打退了。

但她的心情并没有轻松。

因为她知道——

这只是开始。

那只 C 级变异兽，还在五百米外等着。

它还没有出手。

而城墙上的弹药和异能消耗——

已经不少了。

沈暮雪从空间里取出一箱箭矢，让补给组送到城墙上去。

然后她看了一眼城墙外——

兽群在重新集结。

第二波，很快就会来。

而且——

会比第一波更猛。

沈暮雪深吸一口气。

她看了一眼自己的右手。

掌心里，暗红色的火焰安静地蛰伏着。

还没到时候。

再等等。

再等——

城墙上，一个声音突然炸响——

"第二波来了！！西段也发现兽群！！东段也来了！！"

沈暮雪的瞳孔骤缩。

三面合围。

这一次——

是全面的、不计代价的进攻。

C 级变异兽的战术——先用三面合围分散守军兵力，然后集中力量突破一个点。

而那个点——

一定是南段。

因为南段是正对兽群集结方向的位置，承受的压力最大。

"所有人——"韩志远的声音从广播里传来，"三面城墙同时防御！裴将军已前往西段支援！南段坚持住！"

裴夜去西段了。

南段——

没有 S 级坐镇。

沈暮雪握紧了拳头。

她知道这意味着什么。

如果 C 级变异兽选择这个时机攻击南段——

这里没有人能挡住它。

除了——

她自己。

沈暮雪闭上眼，又睁开。

"来吧。"她低声说。

不管你是 C 级还是 B 级——

南段是她的。

谁也别想从这里过去。
