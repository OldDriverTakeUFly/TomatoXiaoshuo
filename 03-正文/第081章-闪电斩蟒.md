# 第八十一章 闪电斩蟒

四路同时进攻。

城墙上的广播在循环播放着战斗指令，士兵和异能者各就各位。

沈暮雪站在南段城墙最高点，目光扫过四个方向——

南面，变异虎正在匀速逼近，八百米、七百米、六百米——

东面，变异蟒像一条绿色的河流在地面上涌动，速度快得不像二十米长的庞然大物。

西面天空，变异鹰在两百米高空盘旋，铁灰色的翅膀在灰蒙蒙的天幕下划出一圈圈弧线。

北面地面，一道道移动的土丘在城墙内侧的地面上缓缓推进——变异蝎正在地下挖掘隧道。

"小豆子——持续报位。"沈暮雪低声说。

"嘤！"

小豆子缩在兜帽里，冰蓝色的眼睛紧盯四方，每隔几秒就用爪子指一下方向。

"变异蟒——东段城墙正前方，距离四百米，速度不变。"

"变异虎——南段正前方，距离五百米，开始加速。"

"变异鹰——高度下降到一百五十米，准备俯冲。"

"变异蝎——地下深度两米，已经进入城墙地基下方。"

沈暮雪把情报通过四路对讲机同步传给每一段的指挥官。

整个基地的防御系统因为小豆子的精确侦察——获得了至少三分钟的预警时间。

在战场上，三分钟——

足够改变一切。

---

东段城墙。

变异蟒第一个到达。

二十米长的巨蟒撞上城墙的那一刻——

"轰——！！"

整面墙都在震。

混凝土碎块从墙顶飞落，铁丝网被巨蟒的身体缠绕碾压，发出刺耳的金属扭曲声。

变异蟒的攻击方式不是撞——是缠。

它用巨大的身体缠绕住城墙，然后收缩——像绞杀猎物一样，要把城墙勒碎。

"开火！"东段指挥官喊。

重机枪和异能同时开火——子弹、风刃、火球——打在蟒身上，但墨绿色的鳞片硬得像铁板，大部分攻击只在上面留下了白印。

"该死——鳞太厚了！"

变异蟒无视了城墙上的攻击，继续收缩——

"咔嚓——"

混凝土在它的绞杀下开始碎裂。

如果让它继续——东段城墙三分钟内就会塌。

就在这时——

天空中落下一道白紫色的闪电。

不是从云层里——是从一个人的手里。

裴夜站在东段城墙的最高点，右手高举——

白紫色的电弧在他的指尖凝聚成了一柄光矛——比上次打变异虎时更粗、更亮、更炽烈。

他没有犹豫。

掷——

"嗤————！！！"

光矛像一道落雷，精准地刺入了变异蟒的头部——

七寸的位置。

蟒的鳞片再硬——也挡不住 S 级的雷系贯穿。

光矛从蟒的头顶穿入，从七寸穿出——

带着一串暗红色的血和焦黑的鳞片碎片。

变异蟒的身体猛地一僵——

缠绕城墙的力量消失了。

二十米长的巨蟒——像一条断了线的绳子——

从城墙上滑落，重重地摔在地上。

它还在动——巨大的身体在抽搐，尾巴疯狂地拍打着地面。

但没有威胁了。

光矛已经摧毁了它的中枢神经。

裴夜从城墙上跳下来，走到蟒的头部旁边——

抽出腰间的刀，一刀斩下了它的头。

彻底死了。

从变异蟒攻城到被杀——

不到四十秒。

秒杀。

东段城墙上的守军——集体看呆了。

他们中的大部分人，这是第一次亲眼看见裴夜出手。

上次兽潮他在南段——很多人只看到了远处的闪电。

这一次——近在咫尺。

S 级的压迫感、光矛贯穿巨蟒的画面、一刀斩首的干脆——

"妈的……"一个士兵喃喃道，"这就是 S 级？"

"人形核弹啊……"

裴夜没有理会他们的议论。

他蹲下来，用刀剖开蟒的头骨——取出一颗拳头大小、泛着墨绿色微光的晶核。

C 级晶核。

他收好晶核，站起来，看了一眼南段方向——

那边，沈暮雪的战斗，即将开始。

"去吧。"他在心里说。

---

南段城墙。

变异虎到了。

五百米——三百米——一百米——

它没有像上次那样先派小兵消耗。

这一次——它亲自冲。

四米长的身体，以 C 级的爆发力，像一颗暗金色的炮弹——

直直地撞向南段城墙。

"嘭——！！！"

城墙震了一下——但没有塌。

霍北的土系加固起了作用——城墙比上次硬了三倍。

变异虎被弹了回去，踉跄了两步，琥珀色的眼睛里闪过一丝意外。

"它没料到城墙变硬了。"陈雪站在沈暮雪旁边，"上次它一掌就拍塌了——这次不行了。"

沈暮雪没有说话。

她看着变异虎。

变异虎也看着她。

一人一虎——隔着城墙，对视。

和上次一样的画面。

但这一次——

沈暮雪不再是那个跪在地上、灵气耗尽的 D 级了。

"陈雪。"她开口。

"嗯？"

"你带战斗队清理它带来的 D 级杂兵。变异虎——交给我。"

陈雪看了她一眼。

"C 级对 C 级——你有把握吗？"

"它有旧伤。"沈暮雪说，"左肩——上次我的压缩火球烧的。裴夜的雷矛也刺过那一侧。旧伤部位的皮甲有裂纹——那是它的弱点。"

"而且——"她抬起右手，金红色的火焰在掌心凝聚，"我是 C 级了。"

陈雪看着那团金红色的火焰——

和上次暗红色的完全不同。

更亮。更烈。更纯粹。

"好。"陈雪点头，"变异虎交给你。杂兵我来。"

她转身，对战斗队下令："所有 D 级以上异能者跟我来——清理杂兵！不要让任何一只变异兽靠近城墙！"

十几个异能者跟着陈雪冲向了城墙外侧——

那里，变异虎带来的几十只 D 级变异犬正在冲击城墙根部。

陈雪出手了——B 级风系。

风刃像割草一样扫过兽群——一只、两只、三只——每一下都精准地切断了变异犬的脊椎。

杂兵交给陈雪——绰绰有余。

城墙上的其他人退后了。

留出了一片空地。

沈暮雪一个人，站在城墙边缘。

面对着城墙下的变异虎。

"又见面了。"她说。

变异虎的琥珀色眼睛——眯了一下。

它认出了她。

上次的猎物。

上次的——火。

但这一次的火——

变异虎感觉到了——不一样了。

颜色变了。温度变了。压迫感变了。

它后退了一步。

不是退缩——是调整。

C 级变异兽的战斗本能告诉它——

这个人类，变强了。

不能像上次那样硬冲。

要——找弱点。

沈暮雪也知道——变异虎不会像上次那样蛮冲了。

它有智商。

C 级对 C 级——

不是拼蛮力。

是拼——

脑子。

"来吧。"沈暮雪低声说。

金红色的火焰在她双手中燃烧。

C 级——

正式开战。
