# 第七十八章 C级

沈暮雪从空间里出来的时候，天刚亮。

她站在板房中央，抬起右手。

金红色的火焰在掌心凝聚——不再是之前那种暗红色的、需要酝酿才能成形的火球。

而是一团稳定的、炽烈的、带着金色边缘的火焰。

温度——

她只是站在那里，周围的空气就开始扭曲了。

桌上的温度计——指针跳到了 45℃。

而这只是她"维持"火焰领域时的被动温度。

如果她全力释放——

800℃。

足以熔化铝、烧穿钢板、把变异兽烤成灰烬的温度。

"嘤！"

口袋里传来小豆子的声音——它感受到了沈暮雪身上灵气的暴涨，兴奋地从兜帽里探出脑袋。

沈暮雪摸了摸它的头。

"雪姐突破了？"

她转过头——姜糖站在门口，眼睛亮晶晶的。

她一定是在门口等了很久——沈暮雪最近十天每天泡灵泉，姜糖虽然不知道具体在做什么，但她知道沈暮雪在"闭关修炼"。

"嗯。"沈暮雪点头，"C 级了。"

"C 级！！"姜糖蹦了起来，"雪姐你太厉害了！"

她冲过来抱住沈暮雪的腰——

然后被沈暮雪身上的残余温度烫了一下。

"呀——好烫！"

"抱歉。"沈暮雪赶紧收敛灵气，体温恢复正常，"刚突破，控制还不稳定。"

"没关系没关系！"姜糖揉着手臂，一点不在意，"C 级啊——那不是和陈队长一个级别了？"

"不一样。"沈暮雪说，"陈队长是 B 级——比我高一级。但我是双系——空间加火系。综合战力——"

她想了想。

"大概和 B 级单系差不多。"

姜糖的眼睛更亮了。

"那雪姐现在是不是基地最强的几个人之一了？"

"差不多吧。"沈暮雪笑了笑，"裴夜还是比我强——S 级。陈雪 B 级。我 C 级双系。"

"前三。"

"嗯。"

---

消息很快传开了。

不是沈暮雪主动说的——是训练场上的人看见她用金红色的火焰练功，惊呆了。

"金红色火焰——那是——C 级？！"

"沈暮雪突破 C 级了？！"

"十天前还是 D 级——这么快就——"

"她本来就是双系——现在 C 级双系——这是怪物啊！"

议论声像风一样传遍了基地。

到中午的时候，连韩志远都知道了。

"沈暮雪突破 C 级了？"韩志远看着裴夜，"你确认？"

"确认。"裴夜靠在墙上，"今早测的。金红色火焰，完整版火焰领域，炎爆技能已解锁。"

韩志远深吸了一口气。

C 级双系。

在青北基地——这是仅次于裴夜（S 级）和陈雪（B 级）的第三强战力。

而且——她才二十三岁。

觉醒异能才两个月。

这个成长速度——

"怪物"都不足以形容。

"好消息。"韩志远说，"因为——"

他的表情凝重了起来。

"第二轮兽潮——三天内会来。规模是第一轮的三到五倍。而且——"

"侦察兵报告——那只被裴将军劈跑的 C 级变异虎，回来了。"

"而且它不是一个人来的。"

"它带了至少三只 C 级同伴。"

沈暮雪的心沉了一下。

四只 C 级变异兽。

第一轮兽潮只有一只 C 级——就让基地损失惨重。

四只——

"基地目前的战力——"裴夜接过话，"我 S 级，沈暮雪 C 级双系，陈雪 B 级。加上十几个 D 级和 E 级的异能者。"

"四只 C 级变异兽——我可以拖住两只。陈雪和沈暮雪联手可以打一只。剩下的一只——"

"交给城防军和 D 级异能者群。"韩志远说，"用人数堆。"

"代价会很大。"裴夜说。

"我知道。"韩志远闭上眼，"但没有别的办法。"

沈暮雪站在那里，听着这两个男人讨论着即将到来的生死之战。

她想起了第一轮兽潮——

一百二十三人死亡。

城墙被破。

她一个人冲上去拦 C 级变异虎，差点死了。

而现在——

四只 C 级。

比上次难四倍。

但这一次——

她不再是那个灵气耗尽、跪在地上无能为力的 D 级了。

她是 C 级。

双系。

有炎爆。

有火焰领域。

有完整版的战斗力。

还有——十天来磨合到极致的狩猎队。

还有——裴夜。

"三天。"沈暮雪说，"三天时间备战。"

她转身走出指挥部。

走了两步，回头看了一眼韩志远和裴夜。

"这次——不会再让城墙被破了。"

---

接下来的三天，基地进入了疯狂备战模式。

城墙的缺口被彻底修补了——不是用铁板和沙袋，而是用霍北的土系异能浇筑了新的混凝土。

霍北一个人干了二十个工兵的活——他在城墙各处预设了"土墙触发点"，一旦兽潮来临，他可以在关键时刻在任何位置升起土墙。

陈雪带着战斗队进行针对性的 C 级对抗训练——用模拟手段让队员熟悉 C 级变异兽的攻击模式。

沈暮雪在训练场练炎爆——她需要精确控制爆炸范围，不能误伤友军。

姜糖在医疗中心囤积了大量灵泉水（稀释的）——作为战时的紧急治疗手段。

小周在后方催生了一批药用植物——止血草、退烧草、消炎草。

宋老把医疗队扩充到了三十人——全员待命。

而裴夜——

他消失了三天。

没有人知道他去了哪里。

第三天傍晚，他回来了。

站在沈暮雪的板房门口。

"我去侦察了。"他说，"C 级变异兽有四只——变异虎、变异蟒、变异鹰、变异蝎。分别在兽群的不同方向。"

"他们的战术是——蟒和虎正面攻城，鹰从空中突袭后方，蝎从地下挖隧道偷袭。"

"四路进攻。"

沈暮雪的眉头紧锁。

地面、空中、地下——三个维度同时进攻。

"最难处理的是鹰和蝎。"裴夜说，"鹰从空中来——大部分异能者没有对空手段。蝎从地下来——城墙挡不住。"

"鹰交给我。"沈暮雪说，"火系——可以打空中目标。"

"蝎呢？"

"霍北。"沈暮雪想了想，"土系可以感知地下的动静。让他在城墙内侧布设土墙阵——蝎一冒头就堵。"

裴夜点头。

"虎和蟒——我来。"

"你能同时打两只 C 级？"

"不是同时。"裴夜说，"先打蟒——蟒是四只里最弱的。秒杀蟒之后——再打虎。"

"虎是上次那只——它的皮甲被你的压缩火球和我的雷矛各破过一次——应该还有旧伤。"

"有旧伤就好打。"沈暮雪说。

两人对视了一眼。

默契——不需要太多言语。

"三天后——"沈暮雪深吸一口气，"我们赢。"

裴夜看着她。

金红色的火光在她眼底跳动——像两团小小的太阳。

C 级。

十天的灵泉浸泡和实战磨练——把她从一个"有潜力的新人"，锻造成了一个真正的战士。

"嗯。"裴夜说。

一个字。

但他的眼神里——

有信任。

有骄傲。

还有一种压了两辈子的、终于得以安放的——

安心。

因为这一次——

他不再是"一个人冲过去来不及"了。

他身边有人。

一个值得他并肩的人。
