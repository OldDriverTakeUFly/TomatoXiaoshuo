# 第八十四章 最后一只

北段城墙内侧。

霍北快撑不住了。

变异蝎被困在土墙阵里——但它的攻击力远超霍北的防御力。

C 级对 D- 级——差了两个大级别。

每一击——霍北的土墙都在碎裂。

他已经修了七层——碎了七层。

灵气消耗过半。

汗水混着泥土糊了满脸，络腮胡上全是灰。

"妈的——这玩意儿力气也太大了吧——"他咬着牙，双手不停地拍地面，一层接一层地升土墙——

"嘭！"

变异蝎的毒刺又穿透了一面——

暗绿色的毒液溅在土墙上——

"滋滋——"

土墙被毒液腐蚀——直接化成了泥浆。

霍北的瞳孔收缩了一下。

毒液能腐蚀土墙——

如果毒液溅到他身上——

"霍大哥——让开！！"

金红色的光从身后射来——

沈暮雪到了。

霍北本能地往侧面一滚——

沈暮雪的火焰领域展开了——

800度的高温笼罩了整个土墙阵的区域——

变异蝎被困在土墙里——无处可逃——被高温直接笼罩——

"嘶——！！"

变异蝎发出了一声尖锐的嘶叫——

它的暗红色甲壳——在高温下开始变色——从暗红变成了暗红发黑——

但——

比变异虎和鹰更抗烧。

蝎子的甲壳天生就是耐高温的——它们在沙漠里生存，甲壳的耐热极限远超其他物种。

即使是800度——它也没有像虎和鹰那样迅速崩溃。

"嘭！"

变异蝎用尾巴猛地一砸——

它面前的土墙——在高温和物理冲击的双重作用下——彻底碎了——

它冲了出来。

直冲沈暮雪。

毒刺——直刺她的胸口。

C 级的全力一击——

沈暮雪没有躲——

她张开双手——

火焰铠甲全力凝聚在胸口——

毒刺撞上了火焰铠甲——

"滋——！！"

金属化的毒刺和金红色的火焰——碰撞在一起——

高温让毒刺的尖端开始发红——

但冲击力还在——

沈暮雪被推着后退了两步——

脚下的地面被犁出了两道沟。

但她——

挡住了。

C 级的火焰铠甲——挡住了 C 级变异蝎的全力一击。

"你——"沈暮雪双手抓住毒刺——灼热的金属在她掌心发出"滋滋"声——

她攥住毒刺——不松手——

变异蝎想抽回尾巴——

但沈暮雪的力气——在灵果和灵泉的双重提升下——大得惊人。

她拽住了变异蝎的尾巴——

然后——

"炎爆——！"

不是远程释放——

是——零距离。

金红色的火球——在她和变异蝎之间的缝隙中——

爆炸。

"轰——！！！"

变异蝎的整个前半身——被爆炸吞没——

暗红色的甲壳在爆炸的高温和冲击波中——

碎了。

像一块被锤子砸中的瓷器——

从头到尾——裂开了一条长长的缝隙。

"嘶————！！！"

变异蝎发出了最后一声——

比之前所有的叫声都凄厉——

然后——

它的八条腿同时软了——

身体轰然倒地。

暗绿色的血液——从甲壳的裂缝中涌出来——

在高温下蒸发——变成一股绿色的烟雾。

沈暮雪松开手——

毒刺的残骸在她掌心留下了一道焦黑的印子。

她站在变异蝎的尸体旁边——

喘着粗气——

金红色的火焰铠甲缓缓消散。

四只 C 级。

蟒——裴夜秒杀。

虎——沈暮雪击杀。

鹰——沈暮雪击杀。

蝎——沈暮雪击杀。

全灭。

沈暮雪弯腰，从变异蝎的头骨中取出晶核。

暗红色。

C 级。

第四颗。

她把晶核收好——

然后——

腿一软——

单膝跪地。

C 级的战斗消耗——即使对她来说——也是巨大的。

四场战斗——火蛇、火箭、火焰领域、炎爆——

她的灵气消耗了七成。

身体——也到了极限。

"沈妹子！"霍北冲过来，扶住她。

"没事。"沈暮雪撑着膝盖站起来，"只是——累了。"

"你太猛了——"霍北一脸又惊又怕的表情，"你一个人杀了三只 C 级——你知道这是什么概念吗？"

"知道。"沈暮雪笑了笑，"回去再算。先——打扫战场。"

城墙上传来了——

震天的欢呼声。

比之前任何一次都响亮。

因为——

四只 C 级——全灭。

青北基地——守住了。

---

战后的清点持续了一整天。

结果——

比第一轮好得多。

死亡：三十一人。

伤员：约一百五十人。

城墙损伤：南段和北段有裂缝，但主体结构完好——霍北的加固起了决定性作用。

物资消耗：弹药六成，箭矢四成，医疗物资五成。

而战果——

四只 C 级变异兽——全部击杀。

C 级晶核四颗——一颗蟒的（裴夜取的）、一颗虎的、一颗鹰的、一颗蝎的。

D 级晶核二十三颗。

E 级晶核五十一颗。

变异兽材料——皮、骨、肉——大量回收。

这个战果——

放在整个末世——都是惊人的。

一只 C 级变异兽就足以毁灭一个小型基地——

而青北一天之内杀了四只。

消息传出后——

不只是青北——

连南域、西疆、东海——所有收到消息的基地——

都震动了。

"青北杀了四只 C 级？！"

"怎么做到的？！"

"听说是一个叫沈暮雪的女异能者——C 级双系——一个人杀了三只！"

"还有裴夜——S 级雷系——秒杀了一只二十米的变异蟒——"

"青北……什么时候变得这么强了？"

沈暮雪的名字——

第一次——

传出了青北基地的围墙。

传向了更广阔的末世。
