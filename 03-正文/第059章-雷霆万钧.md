# 第五十九章 雷霆万钧

裴夜站在沈暮雪和变异虎之间。

他的背影宽阔，像一面墙。

但和城墙不同——这面墙是活的，是带着温度的，是刚才从天而降、用一道闪电劈退了 C 级变异虎的。

变异虎后退了三步，琥珀色的眼睛盯着裴夜。

它的头顶冒着青烟——刚才那一道闪电虽然没能杀死它，但确实打疼了。

暗金色的皮甲上出现了一道裂纹——像瓷器被磕了一道。

这是两天以来，变异虎第一次受伤。

而且是被一个人类——打伤的。

变异虎的琥珀色眼睛里，恼怒消退了，取而代之的是——

警惕。

它开始认真审视面前这个人类。

黑色衣服。宽阔的肩膀。站在那里不动，像一座山。

但变异虎的感知告诉它——

这个人，危险。

极度危险。

裴夜没有看它。

他先低头看了沈暮雪一眼。

她跪在地上，脸色惨白，浑身是汗，灵气彻底耗尽——连站起来的力气都没有了。

但她的眼睛还是亮的。

没有恐惧。没有绝望。

只有一种"你终于来了"的——安心。

"伤到哪了？"裴夜问。

"没伤。"沈暮雪的声音沙哑，"灵气耗光了。"

裴夜的眉心微微拧了一下——极轻微的、旁人根本看不出来的皱眉。

但他什么都没说。

他转回身，面对变异虎。

"退后。"他对沈暮雪说。

沈暮雪咬着牙，挣扎着站起来，踉跄着退到了十米外的废墟后面。

裴夜一个人，站在城墙的缺口处。

面对 C 级变异虎。

身后，是几万个活生生的人。

他抬起右手。

白紫色的电弧在他的指尖跳动——和之前那种随手一击完全不同。

这一次，他的表情是认真的。

真正的 S 级。

全力以赴。

"你的对手——是我。"

裴夜的声音不大，但在寂静的战场上清清楚楚。

变异虎听懂了。

它发出了一声低沉的咆哮——不是之前那种大范围的声波攻击，而是一种蓄力的、凝聚的——

战吼。

它的暗金色皮毛开始发光——真正意义上的发光。

每一根钢针一样的毛发，都泛起了一层暗金色的光泽，像是一套覆盖全身的铠甲。

C 级变异兽的终极防御状态。

然后——

它冲了过来。

四米长的身体，以不可思议的速度——比城墙上的任何人预想的都快——朝裴夜扑来。

速度快到空气都发出了尖锐的啸叫。

变异虎张开獠牙，一爪拍向裴夜的头顶——

这一爪，能把城墙拍碎。

但裴夜——

消失了。

不是真的消失。

是太快了。

S 级的速度，在 C 级的感知范围内——就是瞬移。

变异虎的爪子拍在了空地上——

"轰——！！"

地面被拍出了一个两米深的坑。

碎石飞溅。尘土冲天。

但裴夜已经不在那个位置了。

他出现在了变异虎的侧面——

右手的电弧已经凝聚成了一道光矛。

白紫色的光矛长约两米，尖端的电弧在空气中发出刺耳的"滋滋"声。

空气中弥漫着浓烈的臭氧味。

裴夜没有犹豫——

光矛刺出——

目标是变异虎的左肋——刚才被沈暮雪的压缩火球烧伤的那个位置。

皮甲最薄弱的点。

"嗤——！！"

光矛刺入了变异虎的左肋。

白紫色的雷系灵气像蛇一样钻入它的体内，在它的经脉里疯狂乱窜——

"嗷————！！！"

变异虎发出了一声撕心裂肺的嚎叫。

那声音不是愤怒——

是痛。

真正的、深入骨髓的痛。

它猛地甩动身体，试图把裴夜甩开——

裴夜松开光矛，后跳三米，稳稳落地。

变异虎踉跄着后退了两步。

它的左肋——

光矛刺入的位置——

暗金色的皮甲裂开了。

裂缝里有血渗出来。

暗红色的、带着微弱金色光泽的血——C 级变异兽的血。

它流血了。

真正意义上的受伤。

不是皮外伤——是雷系灵气渗透进了它的体内，正在从内部破坏它的经脉。

"嗷——！"

变异虎怒吼一声，暗金色的皮毛上泛起了一层更浓的光——

它在用体内的灵气，压制裴夜注入的雷系能量。

两股力量在它体内角力——暗金色 vs 白紫色。

僵持了大约五秒。

然后——

变异虎成功地把雷系能量逼了出来。

伤口不再流血，开始愈合。

C 级的恢复力。

裴夜看着这一幕，表情没什么变化。

但他的心里——

微微沉了一下。

这只 C 级变异虎的恢复力，比他预估的强。

一击不够。

需要更多。

但裴夜没有急着出手第二击。

因为——

他不需要杀死它。

他只需要——拖住它。

沈暮雪说拖十分钟。

但裴夜要做的是——拖到兽群退去。

只要兽群失去了 C 级头领的指挥，就会溃散。

变异虎不傻——如果它发现自己打不过裴夜，它会选择撤退。

所以裴夜的策略是——

不求杀，只求压。

用绝对的实力差距，让变异虎意识到——

今天破不了城。

它自然会退。

裴夜举起双手——

十根手指尖，同时亮起了白紫色的电弧。

十道光蛇，在空气中蜿蜒舞动。

整个战场被白紫色的光芒照亮了——

那不是普通的雷电。

那是 S 级雷系异能者的——

领域。

变异虎的琥珀色眼睛里，终于出现了——

恐惧。

---

与此同时。

城墙缺口处。

涌入的变异兽群正在往基地内部冲——

但它们被挡住了。

不是被城墙挡住的——是被火挡住的。

沈暮雪跪在废墟后面，虽然灵气耗尽了，但她还有——

空间。

她从空间里取出了最后的两颗手雷。

这是她在黑市花大价钱买的，一直藏着没用。

现在——

"用不上了最好。"她当时想。

"用上了——就是救命的。"

她拔掉第一颗手雷的拉环，扔向了缺口处涌进来的变异兽群——

"嘭——！！"

爆炸。

手雷的杀伤半径约十五米——缺口处涌入的变异兽被炸倒了一片。

碎石和弹片横飞，堵住了缺口的一部分。

沈暮雪又拔掉第二颗的拉环——

扔了出去。

"嘭——！！"

第二次爆炸。

缺口被碎石和变异兽的尸体彻底堵死了。

后面的变异兽挤不上来。

沈暮雪靠在废墟后面，大口喘气。

手雷用完了。

灵气用完了。

弩箭也射光了。

她现在——

真的什么都做不了了。

只能看着裴夜和变异虎的战斗。

看着那个黑色的背影，站在她面前，用雷电和血肉之躯，挡住了一头 C 级变异兽。

"裴夜……"她低声说。

声音沙哑到几乎听不见。

但她知道——

他会赢。

因为他是裴夜。

北境战神。

S 级雷系。

末世里，站在食物链顶端的人。

---

裴夜的雷系领域展开了。

白紫色的光芒笼罩了方圆三十米的区域——

在这个领域内，空气中充满了雷系灵气。

变异虎每呼吸一次，就有雷系灵气渗入它的体内。

它的暗金色皮甲可以挡住物理攻击和大部分元素攻击——

但挡不住持续的、渗透性的侵蚀。

裴夜不需要攻击它。

他只需要站在那里，维持领域，让变异虎自己感受——

什么叫做"S 级"。

一分钟。

两分钟。

三分钟——

变异虎的呼吸开始急促了。

它的皮甲表面出现了细密的裂纹——不是被打的，是雷系灵气从内部撑出来的。

它的动作变慢了。

它的眼神——从警惕变成了犹豫。

C 级变异兽的智商足够高——它意识到，面前这个人不是它能战胜的。

继续打下去，只有死。

变异虎后退了一步。

裴夜没有追。

他只是站在那里，维持着领域，看着它。

像是一座不可逾越的山。

变异虎又后退了一步。

然后——

它转身了。

四米长的身体，带着一身伤痕和疲惫，朝着城南的方向跑去。

它跑了。

C 级变异虎——跑了。

它身后那些还在城墙外聚集的变异兽群，在头领逃跑的瞬间——

彻底崩溃了。

像退潮一样，黑色的兽群开始向四面八方溃散。

嚎叫声从进攻变成了——逃窜。

城墙上爆发出了一阵震天的欢呼声。

"退了！它们退了！！"

"兽潮退了！！！"

"裴将军万岁！！"

裴夜收回领域。

白紫色的光芒消散。

他站在缺口处，转身走向沈暮雪。

沈暮雪还靠在废墟后面，抬头看着他走过来。

逆着初升的朝阳——

他的身影被金色的光勾勒出轮廓。

像是从画里走出来的人。

裴夜走到她面前，蹲下来。

"还能站起来吗？"

沈暮雪看着他。

她想说"能"。

但她的腿——真的没力气了。

"……可能需要你扶一下。"

裴夜伸出手。

沈暮雪握住他的手，借力站了起来。

她的腿发软，晃了一下——

裴夜的另一只手，扶住了她的腰。

稳稳的。

有力的。

沈暮雪的耳朵又红了。

但这一次，她没有躲开。

"谢谢。"她说。

"不客气。"

裴夜扶着她，一步一步走出了废墟。

晨光落在两人身上。

身后，是废墟和硝烟。

前方，是朝阳和欢呼。

沈暮雪靠在裴夜身上，看着远处的地平线。

兽潮退了。

他们——

活下来了。
