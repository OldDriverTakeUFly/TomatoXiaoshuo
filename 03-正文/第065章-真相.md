# 第六十五章 真相

又过了两天。

陈雪带回了调查结果。

"找到了。"她把一份手写的报告拍在沈暮雪面前，"你的物资没问题——问题出在仓库。"

沈暮雪接过报告，快速扫了一遍。

陈雪调阅了物资运输和入库的全流程记录，发现了几个关键异常：

第一，沈暮雪供应的蔬菜，在入库时的质检记录是合格的——新鲜、无腐烂。

但在出库分发给 C 区住户的前一天，有一批蔬菜被"临时转移"到了另一个仓库。

那个仓库——没有温控设备。

在末世后的高温环境下，蔬菜在那个仓库里放了一天一夜——发霉了。

然后，发霉的蔬菜被"正常分发"给了 C 区住户。

第二，药品的问题更明显——沈暮雪供应的药品入库时，批次号和效期都有登记。但出库时分发的那几盒，批次号对不上。

被掉包了。

"关键证据是——"陈雪指着报告上的一行字，"那个'临时转移'蔬菜的仓库，归谁管？"

沈暮雪看了一眼。

仓库管理员的名字——是一个叫"周强"的人。

"周强是谁？"

"孙建国的手下。"陈雪说，"退役前是武警，末世后跟着孙建国混进基地的。"

铁证。

沈暮雪的物资在入库之后、出库之前，被孙建国的人掉了包——

发霉的蔬菜、过期的药品——全部栽赃到沈暮雪头上。

目的就是——毁掉她的名声。

"证据够吗？"沈暮雪问。

"运输记录、入库质检报告、仓库转移签收单——全都有签字。"陈雪说，"周强的名字白纸黑字写在转移签收单上。"

"赖不掉。"

沈暮雪看着这份报告，心里的大石头落了地。

有了证据——她就能翻盘。

"谢谢陈队长。"

"客气了。"陈雪摆了摆手，"不过——这份证据你现在还不能直接用。"

"为什么？"

"因为孙建国不是傻子。如果你直接在大会上甩出来——他会立刻把周强推出来当替罪羊，说这是'个人行为'，和他无关。"

"那怎么办？"

"你需要——先让孙建国露出马脚。"陈雪说，"让他以为谣言正在起效、沈暮雪即将身败名裂——然后他在放松警惕的时候，一定会留下更大的把柄。"

"到那时候——一网打尽。"

沈暮雪看着陈雪。

这个女人——不只是一名战士。

她还有脑子和耐心。

"明白了。"沈暮雪点头，"我会——先忍着。"

"忍多久？"

"最多一周。"沈暮雪说，"一周后，基地大会——我当众翻盘。"

"好。"陈雪站起来，"这一周——我继续盯着孙建国那边。有动静第一时间通知你。"

她走到门口，停了一下。

"沈暮雪。"

"嗯？"

"裴将军那边——"陈雪犹豫了一下，"你知道他在做什么吗？"

沈暮雪的表情微微变了。

"他在调查我。"她冷冷地说。

"不。"陈雪摇头，"他在保护你。"

沈暮雪愣住了。

"什么？"

"裴将军让军方的情报人员查的不是你——是孙建国。"陈雪说，"他在追查谣言的源头。那个汇报情报的人——是裴将军安排的，专门盯着孙建国和陆衍的。"

沈暮雪的脑子嗡了一下。

"你——你说什么？"

"你以为裴将军在查你？"陈雪看着她的表情，忍不住笑了，"你误会他了。"

"可是——我亲耳听见他说'继续'——那个情报员在汇报我的信息——"

"他让情报员汇报你的信息，是为了判断'哪些信息已经泄露给了孙建国'。"陈雪说，"他需要知道孙建国掌握了你多少情报——才能针对性地保护你。"

"那个'继续'——是让情报员继续查孙建国，不是继续查你。"

沈暮雪站在那里。

整个人僵住了。

她——

误会了？

裴夜不是在调查她——

他是在——保护她？

用她不知道的方式。

在她看不见的地方。

默默地进行着。

就像他在前世想救她但来不及一样——

这一世，他在她不知道的地方，提前清除了威胁。

而她——

她把他当成了和孙建国一样的人。

她躲着他。

对他冷淡。

说"不关你的事"。

"我……"沈暮雪的声音发颤。

"你去找他吧。"陈雪拍了拍她的肩膀，"有些话——说清楚比较好。"

"误会不解开——会变成遗憾。"

沈暮雪站在原地，攥紧了拳头。

然后——

她跑了出去。

---

B 区。

裴夜的住所。

沈暮雪跑到门口，喘着气，敲了门。

三声。

和裴夜一贯的敲门节奏一样——但力度更大，更急促。

门开了。

裴夜站在门口。

他看着沈暮雪——她的头发因为跑步散了，脸上全是汗，眼睛红红的——

像是哭过。

"你——"裴夜的眉头皱了起来。

"裴夜。"沈暮雪喘着气，抬起头看着他。

"陈雪告诉我了。"她的声音在发抖，"你不是在调查我——你是在保护我。"

裴夜沉默了。

"我——"沈暮雪的眼泪终于掉了下来，"对不起。"

"我误会你了。"

"我以为你和她们一样——把我当成目标——"

"但你不是。"

"你从来都不是。"

裴夜看着她哭。

他这辈子——大概没见过沈暮雪哭。

不管是前世还是这一世——她都是那种被打碎了也不会哭的人。

但现在——

她站在他门口，流着泪，跟他说对不起。

裴夜的喉结动了一下。

"进来。"他说。

沈暮雪跟着他走进了房间。

门在身后关上。

裴夜转过身，看着她。

"我不知道你误会了。"他说，声音很低，"如果你不来——我不会解释。"

"为什么不解释？！"

"因为——"裴夜停了一下，"我习惯了。"

"习惯了什么？"

"习惯了——在你看不见的地方做事。"

沈暮雪看着他。

那双冷的、深的、像结冰的湖面的眼睛——

此刻，冰面下面翻涌着的东西，她终于看清了。

不是冷漠。

是——

克制。

他一直在克制。

从重生那天起，他就习惯了在她的背后默默守护——不打扰她、不给她增加负担、不让她知道自己在做什么。

因为他觉得——"她需要的我时候，我会在"就够了。

他不需要解释。

不需要邀功。

不需要让她知道。

但——

"裴夜。"沈暮雪擦了一把眼泪，认真地看着他。

"我不要你在看不见的地方保护我。"

"我要你知道——"

"你做的事，我都想知道。"

"你不用一个人扛。"

裴夜看着她。

很久。

很久。

然后——

他做了一个沈暮雪从未见过的动作。

他低下头。

额头轻轻抵在了沈暮雪的头顶上。

很轻。

像是在碰一件珍贵的东西。

"好。"他说，声音低到像叹息。

"以后——你知道。"

沈暮雪闭上了眼。

感觉到他的额头抵在自己头顶的温度。

热的。

带着一种她从未在任何人身上感受过的——

安心。
