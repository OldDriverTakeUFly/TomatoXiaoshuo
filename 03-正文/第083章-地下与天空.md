# 第八十三章 地下与天空

变异虎被击杀的消息——像一道闪电——传遍了整个战场。

城墙上、城墙下、东段、西段——所有人都听见了那声最后的嚎叫，和之后震天的欢呼。

"变异虎被沈暮雪杀了！！"

"C 级！她一个人杀了 C 级变异虎！！"

"金红色的火——太猛了——一整个爆炸——变异虎直接被烧成焦炭——"

士气——瞬间暴涨。

城防军和异能者们像是被打了一针强心剂，输出力度陡然提升。

东段的杂兵在裴夜秒杀蟒蛇后本就溃不成军，现在更是疯狂逃窜。

南段的杂兵在陈雪的带领下被清理了大半。

但——

战斗还没有结束。

因为——

还有两只 C 级。

变异蝎——从地下来的。

变异鹰——从天空来的。

---

北段城墙内侧。

霍北蹲在地上，双手按着地面。

他的土系异能——让他能"听"到地下的动静。

"嘤——"他学着小豆子的样子，低声自言自语，"有东西在下面动。"

他闭着眼，全神贯注地感知地下的震动。

变异蝎在城墙地基下方——深度约两米——正在挖掘隧道。

它的目标很明确——挖穿地基，从城墙内侧冒出来，直接攻击后方的人员和设施。

"来了——"霍北睁开眼，"距离冒头点——十米——八米——五米——"

他猛地站起来，双手重重一拍地面——

"土墙阵——起！"

三十六个预设的触发点同时激活——

城墙内侧的地面上，一面接一面的土墙从地下升起——形成了一个密不透风的网格状防御工事。

就在土墙升起的下一秒——

"嘭！！"

地面炸开了——

一只体型如小型轿车的变异蝎从地下钻了出来。

它的甲壳是暗红色的，八条腿像钢柱一样粗，尾巴末端的毒刺闪着暗绿色的光泽——那是剧毒。

如果它的毒刺刺中一个人——

C 级的毒素会在三十秒内杀死一个 B 级异能者。

变异蝎从地下冒头——

发现自己被三面土墙包围了。

"嘿嘿——"霍北站在土墙外面，络腮胡上沾着泥，笑得像只偷了鸡的狐狸，"出来啦？"

变异蝎的猩红眼睛盯着他——

然后——它的尾巴猛地一甩——

"嗖——！"

毒刺像一支标枪一样射向土墙——

"嘭！"

土墙被毒刺穿透了一个洞——但没塌。

霍北的土墙是 D- 级的——挡不住 C 级的全力攻击太久——

但他不需要挡太久。

他只需要——堵住它。

让变异蝎困在地下出口处——出不来。

"再加一层！"霍北一跺脚——第二面土墙在第一面后面升起——

变异蝎想往前冲——被第一面墙挡住。

想往侧面绕——被网格状的土墙挡住。

想退回地下——

霍北早就把出口也堵上了。

"你就在里面待着吧！"霍北拍着手，"想出来——先问过老子的土墙！"

变异蝎被困住了。

它疯狂地攻击土墙——毒刺、钳子、身体冲撞——

土墙在碎裂——但霍北在不停地修补——

D- 级的土系 vs C 级的变异蝎——

霍北打不过它，但他可以——

拖。

拖到有人来帮忙。

---

与此同时——

天空中。

变异鹰俯冲了。

它选择了变异虎被杀、所有人注意力都在地面的时机——

从一百五十米高空，以接近九十度的角度——

直直地俯冲向城墙后方的医疗站。

目标——姜糖。

它知道姜糖在那里。

因为——

方明远告诉了南域。南域告诉了变异兽群的头领。

C 级变异兽之间的通讯——是通过灵气波动的。

四只 C 级在进攻之前就约定好了——鹰的目标是后方的治愈系。

杀掉治愈系——青北就失去了战场医疗能力。

"鹰来了！！"城墙上的瞭望手尖叫——

"高度一百——五十——三十——它在冲医疗站！！"

医疗站里。

姜糖正在给一个伤员包扎——

听见了外面的惊呼声。

她抬头——

看见一个巨大的、铁灰色的影子——从天空中砸下来——

像一颗铁做的陨石。

"姜糖——跑！！"张姐冲进来，一把拉住她——

但来不及了——

变异鹰的速度太快了——从一百五十米俯冲到地面——只需要三秒。

三秒——

她们跑不出医疗站。

"嘭——！！"

一个东西砸在了医疗站的屋顶上——

铁皮屋顶被砸穿了一个大洞——

但不是变异鹰。

是——

一团金红色的火焰。

沈暮雪。

她从城墙上跳下来——不是往城外跳——是往城内跳——往医疗站方向跳。

她在变异鹰俯冲到三十米的时候——就发现了它的目标。

然后她用空间系——从城墙直接转移到了医疗站上方——

在变异鹰撞穿屋顶的前一秒——

释放了火焰领域。

800度。

变异鹰撞进火焰领域的瞬间——

它的铁灰色羽毛被高温灼烧——开始卷曲、发红——

"嘎——！！"

变异鹰发出了一声刺耳的尖叫——

它的羽毛是金属化的——硬度堪比钢板——

但金属——导热。

800度的高温通过金属化的羽毛——直接传导到了它的皮肤和肌肉上。

变异鹰的整个身体——在火焰中被"烤"了。

它疯狂地拍打翅膀——试图飞起来——逃离这个灼热的区域——

但沈暮雪不让它走。

和变异虎一样——

不能让它跑了。

跑了——还会回来。

她双手一推——

"炎爆——！"

金红色的火球飞出——

在变异鹰的身上爆炸——

"轰——！！"

医疗站的屋顶——彻底塌了——

但姜糖和张姐已经被沈暮雪提前用空间转移到了十米外——

她们看着医疗站的废墟中——

一团金红色的火焰——

裹着一只铁灰色的巨鹰——

在燃烧。

"嘎——……"

变异鹰的叫声——越来越弱。

它的金属化羽毛——在800度的持续高温下——开始融化。

铁灰色的金属液滴——从它的身上滚落——滴在地上，发出"滋滋"的声响。

然后——

变异鹰倒了。

它的翅膀——融化了半边。

身体——被烧成了焦黑色。

沈暮雪站在废墟中央——

金红色的火焰铠甲还在燃烧。

她弯腰，从变异鹰的头骨中取出了一颗晶核。

铁灰色。

C 级。

第三只。

三只 C 级——蟒、虎、鹰——全部击杀。

只剩——

变异蝎。

还被霍北困在土墙里。

"霍大哥——"沈暮雪通过对讲机喊，"还能撑住吗？"

"能——但土墙快碎了——它太猛了——"

"我来了。"

沈暮雪转身——朝北段跑去。

金红色的火焰在她身后拖出一道长长的尾迹——

像一颗流星——

划过基地的天空。
