# 第五十八章 火降

C 级变异兽冲到了城墙前方两百米。

沈暮雪终于看清了它的全貌。

那是一只变异虎。

不——比虎大。

肩高近两米，体长超过四米，浑身覆盖着暗金色的皮毛，每一根毛都像钢针一样竖起。它的额头正中，有一道深色的纹路——不是老虎的"王"字，而是一个更复杂的、像某种远古图腾的标记。

它的眼睛是琥珀色的，不是普通变异兽的猩红。

琥珀色的眼睛里，没有疯狂，没有嗜血——

只有冷。

一种居高临下的、审视蝼蚁的冷。

它站在两百米外，看着城墙。

看着城墙上那些渺小的、颤抖的人类。

然后——

它吼了。

"嗷————！！！"

这一声，不是之前那些变异犬的嚎叫。

这是一种——

声波。

物理层面的冲击。

沈暮雪感觉自己像是被一面无形的墙撞了一下——耳朵嗡嗡响，胸口发闷，腿一软差点跪下去。

城墙上的人更惨——很多普通士兵直接被震晕了，异能者也好不到哪去，一个个脸色惨白，灵气运转紊乱。

"这是……咆哮技能！"陈雪咬着牙，勉强站稳，"C 级变异兽的领域技能！它能压制范围内所有低等级生物的灵气！"

灵气被压制了。

沈暮雪试着调动火系——果然，原本顺畅的灵气流转变得滞涩了，像是在浓稠的泥浆里游泳。

她的 D 级灵气——在 C 级的领域压制下——只剩下平时的六成。

六成。

面对一只 C 级变异兽。

形势——

极度严峻。

变异虎发出咆哮之后，开始向城墙走来。

不是跑。

是走。

它甚至没有加速——因为它不需要。

它是 C 级。

在它面前，城墙上的这些人——不过是摆在盘子里的食物。

变异虎走到城墙下方——

它的头，和城墙顶平齐。

琥珀色的眼睛扫过城墙上的守军，最后——

落在了沈暮雪身上。

它认出了她。

今天用火烧了它无数手下的那个女人。

火系异能者。

变异虎的嘴角微微咧开——

露出了两排白色的、像匕首一样的獠牙。

它在笑。

沈暮雪的后背瞬间被冷汗浸透了。

她不怕变异兽。

但面对一只 C 级的、明显拥有高智商的变异虎——

她的每一根神经都在尖叫着"危险"。

但——

她没有退。

"陈队长。"她按下对讲机，声音出奇地稳。

"在。"

"撤走城墙上的普通人。留下异能者。"

"你——"

"普通人面对 C 级只有送死。"沈暮雪说，"让他们退到第二防线。异能者留下来——拖住它。"

"拖到什么时候？"

"拖到裴夜来。"

陈雪沉默了一秒。

然后——

"所有非异能者！撤到第二防线！快！"

命令下达，城墙上的普通士兵如蒙大赦，开始有序撤离。

留下来的——只有异能者。

大约三十个人。

D 级三个（包括沈暮雪），E 级十几个，F 级十几个。

面对一只 C 级变异虎。

兵力对比——

悬殊到了可笑的程度。

但没有人跑。

因为这三十个人都知道——

如果他们退了，C 级变异虎就会突破城墙，涌入基地。

几万人——老人、孩子、伤员——

全部会死在这里。

他们退不了。

沈暮雪站在城墙的最前沿。

变异虎就在五米下方，琥珀色的眼睛和她的眼睛平齐。

一人一兽对视。

"你是来破城的？"沈暮雪低声说。

变异虎当然不会回答。

但它歪了一下头。

像是在说——"是。"

"那——"

沈暮雪举起右手。

掌心里，仅剩的三成灵气全部涌出——

不是火球。不是火蛇。

是她目前能释放的、最大威力的攻击——

压缩火球。

D 级的极限。

一颗拳头大小的、炽白色的火球，在她掌心凝聚。

温度高到空气都在扭曲，城墙上的混凝土开始发烫。

"那就从我开始——"

"看看你能不能过去。"

她掷出火球——

炽白色的火球呼啸着飞向变异虎的头部——

变异虎的琥珀色眼睛微微眯了一下。

然后——

它抬起了一只前爪。

漫不经心地——

拍了下去。

就像人拍苍蝇一样随意。

"嘭——！"

火球被它的爪子拍中了。

炽白色的火球在它的掌心炸开——火焰四溅，灼热的气浪席卷了整个城墙正面。

但——

变异虎的爪子毫发无伤。

它的皮毛上泛起了一层暗金色的微光——那是它的变异皮甲，硬度堪比钢铁。

D 级的压缩火球——

连它的皮都没烧穿。

沈暮雪的瞳孔骤缩。

差距太大了。

D 级和 C 级之间——不是"差一级"那么简单。

是质的飞跃。

就像蚂蚁和大象的区别。

"嗷——"

变异虎低吼了一声，前爪猛地拍向城墙——

"轰——！！"

五米高的混凝土城墙——

裂了。

不是出现裂缝那种裂。

是从被拍的位置开始，像蛛网一样向外扩散的大面积碎裂。

混凝土碎片飞溅。

城墙在塌。

"快退！"陈雪一把拉住沈暮雪，往后跑。

两人刚跑开——

"轰隆——！！"

那段城墙——塌了。

整段五米高、十米宽的墙体，在变异虎的一掌之下——

轰然倒塌。

尘土飞扬。

碎石滚落。

城墙——

被破了一个十米宽的缺口。

变异虎站在缺口处，琥珀色的眼睛扫过基地内部。

它踏出了一步。

踏入了基地。

"不——"

沈暮雪被陈雪拉着跑了二十米，回头看了一眼——

变异虎进入了基地。

在它身后，那些一直在城墙外等待的变异兽群——

像潮水一样涌向缺口。

城破了。

如果不想办法阻止——

基地几万人——

将面临灭顶之灾。

沈暮雪挣开陈雪的手。

"你干什么！"陈雪喊。

"拦住它。"沈暮雪说。

"你疯了？你是 D 级！它一掌就能拍死你！"

"我知道。"沈暮雪看着她，"但如果不拦住它——所有人都会死。"

"裴夜会来——"

"裴夜在西段！他赶过来至少要十分钟！"沈暮雪的声音陡然提高，"十分钟！十分钟里这只C级能把半个基地屠光！"

"我们——"

"我们不需要杀死它。"沈暮雪说，"只需要拖住它。拖十分钟。"

她转身，面对着正在大步深入基地的变异虎。

她的灵气只剩不到两成了。

但她没有选择。

"陈队长——带着所有人退到第二防线。保护伤员和平民。"

"你——"

"我一个人拖住它。"

"沈暮雪！你——"

沈暮雪没有再听。

她跑了出去。

朝着变异虎的方向。

一个人。

D 级火系。

灵气不到两成。

面对一只 C 级变异虎。

她不知道自己能不能撑过十分钟。

但她知道——

如果她不冲上去，那些伤员、那些平民、那些她一路从青城护到青北的人——

姜糖、霍北、宋老、张姐、小周、小刘和她的女儿——

全部会死。

她不允许。

"来啊——！！"

沈暮雪在奔跑中举起双手，把体内所有剩余的灵气——全部引爆。

暗红色的火焰从她双手中涌出，在她身周形成一道——

火焰旋涡。

这不是火球。不是火蛇。不是压缩火球。

这是她从来没有尝试过的、把所有灵气一把梭哈的——

最后一击。

火焰旋涡裹着她的身体，向变异虎冲去——

变异虎停下了脚步。

琥珀色的眼睛里，终于闪过一丝——

意外。

不是恐惧。

是意外。

像是一个人类，看见一只蚂蚁举起了比自己还大的石头。

"嘭————！！！"

火焰旋涡撞上了变异虎。

爆炸。

灼热的气浪席卷了方圆五十米。

地面被烤裂了。空气被烧扭曲了。缺口处的碎石被高温融化成了琉璃。

火焰的亮度让所有人都不得不闭上眼——

整个南段战场，在那一瞬间，亮如白昼。

然后——

火焰散去。

硝烟弥漫。

沈暮雪跪在地上，双手撑着地面，大口大口地喘气。

她的灵气——彻底耗尽了。

浑身上下一点力气都没有，连抬手的力气都没有了。

她抬起头，透过烟尘看向前方——

变异虎还站着。

它的暗金色皮毛被烧焦了一片——左前肩的位置，有一块巴掌大的区域，皮毛烧光了，露出了底下暗金色的皮甲。

受了伤。

但——

只是皮外伤。

对 C 级来说，几乎可以忽略不计。

变异虎低头看了看自己肩膀上的烧伤，然后抬头看向沈暮雪。

琥珀色的眼睛里，多了一分——

恼怒。

被一只蚂蚁弄疼了。

它张开嘴，露出了獠牙——

朝沈暮雪走过来。

沈暮雪跪在地上，看着它逼近。

她动不了了。

灵气耗尽，体力也耗尽了。

她甚至站不起来。

"拖了……三分钟。"她喃喃自语，"不够……"

变异虎走到了她面前五米的位置。

举起了前爪——

那一爪下去，能把整段城墙拍碎——

更别说一个人。

沈暮雪看着那只落下的爪——

她没有闭眼。

"裴夜……"她在心里说。

然后——

"轰————！！！！"

一道白紫色的闪电。

从天而降。

精准地劈在变异虎的头顶。

变异虎的整个身体被雷光包裹——

它发出了一声从未有过的、痛苦的嚎叫——

"嗷————！！！"

前爪的攻击被打断了。

它踉跄着后退了三步。

沈暮雪抬起头。

一个黑色的身影，从天而降，落在了她和变异虎之间。

黑色长袖。

宽阔的背影。

左手腕上的旧疤。

裴夜。

他来了。

"说了——"他的声音很低，低到只有沈暮雪能听见。

"你需要的我时候，我会在。"
