# 第五十七章 裂缝

第二天。

兽潮第二轮来了。

比第一天更猛烈。

C 级变异兽不再只用人海战术——它亲自指挥，将兽群分成三组：第一组佯攻南段正面，第二组从侧面包抄，第三组——专门挖墙根。

昨天沈暮雪烧堵了一个洞，但今天，变异犬在城墙的不同位置同时开挖。

堵了一个，又冒出两个。

"该死的——它们学聪明了！"城墙上的土系异能者骂道。

沈暮雪站在高台上，用小豆子的寻宝天赋持续侦察战场——

小豆子缩在她的外套兜帽里，只露出半个脑袋，冰蓝色的眼睛盯着城墙外，时不时伸出爪子指方向。

"左前方，墙根，三只在挖。"

"右侧三十米，有埋伏。"

"正前方五十米——有一只 E 级的混在普通变异犬里。"

沈暮雪把情报源源不断地通过对讲机传给陈雪。

陈雪的指挥效率因此提升了一大截——她不需要再猜测兽群的意图，沈暮雪的情报就是她的眼睛。

但即使如此——

城墙还是扛不住了。

不是防御的问题。

是城墙本身的问题。

青北基地的围墙是末世后一个月内紧急修建的——混凝土浇筑，工期赶得紧，质量并不完美。

在几百只变异兽持续两天的冲击下——

墙体开始出现裂缝。

"南段城墙中段——裂缝！宽度五厘米！还在扩大！"

一个工兵跑来报告，脸色惨白。

沈暮雪心里一沉。

裂缝。

如果墙体崩塌——

那就是缺口。

变异兽会从缺口涌入。

"所有人——集中加固南段中段！"陈雪在城墙上吼。

土系异能者冲过去，用灵气修补裂缝——但裂缝太长了，几个人根本修不过来。

"嘭——！"

一声闷响。

裂缝处——混凝土掉了一块。

一个碗口大的洞出现在墙体上。

透过洞口，能看见外面密密麻麻的变异兽——猩红的眼睛、白色的獠牙、灰黑色的毛。

它们看见洞了。

嚎叫声骤然升级——

变异犬群发了疯一样往洞口冲。

"堵住！快堵住！"陈雪喊。

一个土系异能者冲上去，用身体堵住了洞口——

"嘭！"

外面的变异犬撞在洞口上，巨大的冲击力把那个土系异能者撞得倒退了两步。

他咬着牙，双手撑着洞壁，灵气全开——

土墙从他的手掌中生长出来，一点一点地填补着洞口。

但变异犬的挖掘和撞击一刻不停——

他撑不了多久。

"我去！"沈暮雪从高台上跳下来。

她冲到洞口旁边，右手一抬——

火蛇从掌心飞出，穿过洞口，烧向外面的变异犬——

"嗷——！！"

惨叫声。

外面的变异犬被火蛇扫退了十几米。

洞口的压力骤减。

那个土系异能者趁机把洞口修补了大半。

"谢、谢谢……"他喘着气。

"别谢，继续加固。"沈暮雪说。

她站在洞口旁边，维持着火蛇的输出——

但灵气在飞速消耗。

她已经连续战斗了两天。

D 级的灵气储备，在这种高强度的消耗下，正在以肉眼可见的速度下降。

"还有多少？"她问自己。

大约——三成。

三成灵气，撑不了多久了。

而那只 C 级变异兽——

小豆子突然"嘤"了一声，爪子指向远处。

沈暮雪顺着它指的方向看去——

五百米外。

那个巨大的影子——

动了。

C 级变异兽。

它终于亲自出手了。

沈暮雪的心脏猛地收缩了一下。

它来了。

那个一公里外蛰伏了两天的、一直在用兽群消耗守军体力和弹药的——

真正的 Boss。

它选择了这个时机——城墙出现裂缝、守军疲惫不堪、弹药所剩无几——

来收割了。

"陈队长！"沈暮雪按下对讲机。

"收到！"

"C 级变异兽——出动了。正从南面方向逼近。"

对讲机那头沉默了一秒。

然后陈雪的声音响起——带着一种决绝的冷静。

"所有人——准备迎接 C 级。"

"这不是演习。"

"生死——在此一举。"

城墙上的气氛凝固了。

所有人——士兵、异能者、甚至补给组的普通人——

都感觉到了那种压迫感。

大地在微微震动。

那是 C 级变异兽奔跑时的脚步声。

一步。

两步。

三步——

越来越近。

越来越重。

沈暮雪站在洞口旁边，看着南面的方向。

她的灵气只剩三成。

但她知道——

接下来这一战，她不能退。

南段城墙是她的阵地。

这只 C 级变异兽——

她必须挡住。

即使挡不住——

也要拖到裴夜赶来。

"来吧。"她低声说。

掌心里，最后的三成灵气凝聚成暗红色的火焰。

等着——

被释放。
