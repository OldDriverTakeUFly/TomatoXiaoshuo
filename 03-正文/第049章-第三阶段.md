# 第四十九章 第三阶段

入城第十五天。

基地的广播响了。

青北基地的广播系统是用军用级别的设备搭建的——覆盖整个基地，声音清晰洪亮。

平时广播只用来通知日常事务——"今日物资分配时间""B 区异能训练安排""医疗中心疫苗接种通知"之类。

但今天，广播的语气不一样。

"全体基地人员注意。全体基地人员注意。"

"紧急通知：经基地指挥部确认，第三阶段灾变——大规模异兽潮——将于近期爆发。"

"重复一遍：第三阶段异兽潮即将来临。"

"所有异能者即日起进入一级战备状态。所有非战斗人员限区域活动，不得擅自靠近围墙区域。"

"各区域负责人立即前往 A 区指挥部参加紧急会议。"

广播重复了三遍。

沈暮雪站在板房门口，听完了全部内容。

第三阶段。

来了。

前世的记忆告诉她，第三阶段的异兽潮和之前的小打小闹完全不是一个级别。

前两个阶段——极热天灾和初步变异——只是"热身"。

第三阶段才是末世真正的屠杀期。

变异兽会以惊人的速度进化，形成大规模的兽潮，向人类聚集地发起进攻。

前世的青北基地，在第三阶段兽潮中，损失了近三分之一的人口。

那是一场血战。

沈暮雪深吸一口气，转身回到房间。

她从空间里取出所有的武器装备——弩、箭矢、刀、工兵铲、防刺服——逐一检查。

又取出了那颗 D 级火系晶核——这几天她一直在缓慢吸收，火系已经稳定在 D 级中段。

如果兽潮来临，她需要更强的实力。

"雪姐？"姜糖推门进来，脸色有点紧张，"广播说——兽潮要来了？"

"嗯。"沈暮雪点头，"你怕吗？"

姜糖咬了咬嘴唇。

然后她做了一件让沈暮雪意外的事——

"雪姐，我好像……觉醒了。"

沈暮雪转头看她。

姜糖摊开手掌——

一团柔和的、淡金色的光芒，在她的掌心里缓缓凝聚。

不是火。不是土。不是雷。

是——

光。

治愈的光。

"我昨天晚上就感觉身体不对劲……手心一直发热……今天早上起来，发现……"姜糖的声音在发抖，"我能……治东西了。我把一片枯叶子捂在手里，它就……变绿了。"

沈暮雪看着她掌心里那团淡金色的光，呼吸停了一拍。

治愈系。

觉醒了。

前世的记忆成真了——姜糖确实是治愈系。

而且这一次，因为灵泉水的长期调理，她的觉醒质量比前世更好。

"姜糖。"沈暮雪走过去，握住她的手。

那团金色的光在两人交握的手心里流淌，温暖、柔和、充满生机。

"你听我说。"沈暮雪的声音很严肃，"治愈系是末世里最稀有、最珍贵的异能。"

"我知道……"

"不，你不知道。"沈暮雪看着她，"治愈系意味着——无数人会盯着你。有人会想拉拢你，有人会想控制你，甚至有人会想——"

她没有说"抢走你"。

但姜糖读懂了。

"雪姐……"她的声音带了哭腔，"那我怎么办？"

"你听我说完。"沈暮雪握紧了她的手，"从现在开始，你的异能不对外公开。除了我们团队内部，谁也不知道你是治愈系。"

"好……"

"登记的时候，我会帮你处理。"沈暮雪已经在想怎么操作了——她可以找宋老帮忙，把姜糖的异能登记为别的低关注度的系，比如植物系或者辅助型光系。

治愈系的核心信息——隐藏。

"还有——"沈暮雪看着姜糖的眼睛，"不管发生什么事，你在我身边。我不会让任何人把你带走。"

前世姜糖被大势力强行带走，消失在末世的洪流中。

这一世——

沈暮雪攥紧了拳头。

她绝不允许这种事再发生。

"记住了。"她说，"你是我的团队的人。谁想动你——"

她没有说完。

但那双冷冽的眼睛里，有一团暗红色的火焰在跳动。

姜糖看着她，眼泪在眼眶里打转，但嘴角却弯了起来。

"嗯。"她用力点头，"雪姐，我相信你。"

---

当天下午，沈暮雪参加了基地的紧急会议。

作为物资合作方和B区有贡献的异能者，她有资格列席。

会议在A区指挥部的大厅里举行。

到场的有三十多人——管委会的全体成员、军方的主要军官、以及基地里有影响力的异能者代表。

赵德明主持。

韩志远做情况通报。

"各位，情况比我们预估的严重。"韩志远站在地图前，表情凝重。

"过去三天，基地外围的侦察部队发现——变异兽的数量在急剧增加。而且——出现了新的品种。"

他在地图上标注了几个位置。

"城北方向，发现了一群约三百只的变异野犬群，领头的疑似 D 级。"

"城东方向，发现了变异蛇的踪迹——等级不明，但已经造成两名侦察兵牺牲。"

"最严重的——"

他的手指点在地图上的一个位置。

"城南方向，距离基地约十五公里处，发现了一只——C 级变异兽。"

会场里一片倒吸冷气的声音。

C 级。

到目前为止，基地击杀的最高等级变异兽是 D 级——而且还是出动了一整个战斗小队才勉强拿下的。

C 级——那是基地目前的战力上限。

如果这只 C 级变异兽带着兽群来攻——

"根据分析，"韩志远继续说，"第三阶段的兽潮将在七到十天内到来。规模可能是之前的五到十倍。"

"基地需要做好最坏的准备。"

会场沉默了。

每个人的脸上都写满了凝重。

沈暮雪坐在角落里，听完了全部通报。

她的表情很平静。

但心里——

已经开始盘算了。

兽潮要来了。

更大的兽潮。

这一次不是青城那栋楼的小打小闹——

是整个青北基地的生死之战。

她需要做好准备。

让团队做好准备。

让小豆子——那只刚出生不久的灵狐——做好准备。

因为它虽然还是幼体，但它的寻宝天赋和灵气感知能力，在兽潮来临时可能有意想不到的作用。

沈暮雪站起来，走出了会议室。

阳光照在她脸上。

她眯了眯眼，看着远处高耸的围墙。

围墙外面，荒野在热浪中扭曲。

而更远的地方——

兽潮，正在集结。

沈暮雪深吸一口气。

来吧。

这一次，她不是一个人在战斗。

她有一个团队。

有一个基地。

有并肩作战的战友。

还有——一个在暗处守望的男人。

她握紧了拳头。

掌心里，暗红色的火焰，安静地跳动。

等着被释放。
