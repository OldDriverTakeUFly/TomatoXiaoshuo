# 第四卷：域外·征途｜章节细纲

> **卷目标**：远征昆仑遗迹 → 发现域外文明真相 → 关闭第一台灵气装置 → 全球远征（五大洲五台） → SS级突破 → 最终决战
> **字数预估**：约 25 万字 / 35 章（Ch106-140）
> **核心节奏**：冒险+解谜+战斗三线交织，格局从"国内基地争霸"升级到"全球文明存亡"

---

## 段落一：昆仑远征（第 106–115 章）
**🎯 任务**：团队出发 → 路上冒险 → 抵达昆仑遗迹 → 第一台装置关闭 → A级巩固

| 章 | 标题（暂定） | 情节 / 爽点 |
|----|------|-------------|
| 106 | 出征 | 远征队出发。沈暮雪+裴夜+霍北+姜糖+陈雪+小豆子+小银。七个成员踏上昆仑之路。凛冬中出发，目标三千里外的昆仑山脉。 |
| 107 | 废墟之城 | 经过第一座废墟城市——昔日省会，如今被冰雪覆盖。在城市里发现了一个幸存者地下避难所，收编了几个人。温情线。 |
| 108 | 冰裂谷 | 远征队进入一片冰川裂谷地带。小豆子的寻宝天赋在冰层下发现了大量晶核矿脉——意外收获大量高等级晶核。 |
| 109 | 域外生物 | 第一次遭遇域外生物——不是变异兽，是来自裂缝另一侧的东西。形态诡异、不惧冰火、异能抗性极高。沈暮雪用极焰才将其击杀。**新威胁登场。** |
| 110 | 裴夜的推断 | 裴夜分析域外生物的特征——它们是"灵气装置"开启后，从另一个维度渗透过来的。如果不关闭装置，域外生物会越来越多，最终完全取代地球生物。 |
| 111 | 小豆子的共鸣 | 越接近昆仑遗迹，小豆子反应越强烈——它的灵狐血脉与遗迹的灵气产生共鸣。角完全长成，体型暴涨到大型犬，战力突破C级。**萌宠进化。** |
| 112 | 昆仑脚下 | 远征队抵达昆仑山脉脚下。裂缝在山腰处清晰可见——紫色的光从裂缝中渗出。空气中的灵气浓度是外界十倍。 |
| 113 | 遗迹入口 | 裴夜凭借前世记忆，找到了遗迹入口——一个隐藏在冰川下的、远古文明建造的隧道。进入隧道。 |
| 114 | 远古文明 | 隧道内部——巨大的地下空间，布满了远古文明的建筑遗迹。壁画记录了文明的兴衰——他们也曾遭遇灵气灾难，建造了装置来控制灵气，但最终被装置反噬灭亡。 |
| 115 | 第一台装置 | 在遗迹最深处——第一台灵气装置。巨大的、由不明金属构成的、正在运转的机器。灵气从装置中心源源不断地涌出。沈暮雪走上前——准备用极焰关闭它。**段落一高潮收束。** |

## 段落二：关闭装置（第 116–122 章）
**🎯 任务**：关闭第一台装置 → 获得远古信息 → SS级初窥 → 发现全球五台的联动机制

| 章 | 标题 | 情节 / 爽点 |
|----|------|-------------|
| 116 | 极焰·关机 | 沈暮雪用极焰攻击装置核心。装置的防护层在极焰的热胀冷缩下碎裂。但——装置不是简单的机器，它有"自我修复"功能。需要持续输出极焰。**B级不够，需要突破。** |
| 117 | SS级 | 在灵气浓度百倍的遗迹中，沈暮雪吸收大量远古灵气，突破SS级。紫金色极焰变成纯白金色——"灭世极焰"。一击关闭了装置。**🔥 SS级突破大爽点。** |
| 118 | 装置的回应 | 装置关闭瞬间——释放了一段全息信息。远古文明留下的警告：五台装置联动，关一台会让其他四台加速运转。必须在一年内关闭全部五台——否则剩余装置会引发"灵气大爆发"，摧毁地球。**倒计时开始。** |
| 119 | 全球远征计划 | 沈暮雪决定——组建五支远征队，同时前往五大洲关闭装置。她负责最近的一台（昆仑已关，下一台在东南亚）。其他四台——委托给其他基地的A级异能者。 |
| 120 | 回归青北 | 远征队返回青北。SS级的消息震动全球所有基地。沈暮雪召开"全球联盟"会议——视频连线各大洲的幸存者基地。 |
| 121 | 联合 | 各大洲基地响应。西欧、北美、南美、非洲——各派出一支队伍。沈暮雪协调资源、情报和路线。**格局升至全球。** |
| 122 | 分头行动 | 五支队伍分头出发。沈暮雪带自己的团队去东南亚。裴夜去西欧。第一次分别——各自独立执行任务。**CP分离张力。** |

## 段落三：东南亚·第二台（第 123–130 章）
**🎯 任务**：沈暮雪独立领队 → 东南亚遗迹 → 域外生物拦截 → 关闭第二台 → 裴夜线交叉

| 章 | 标题 | 情节 / 爽点 |
|----|------|-------------|
| 123 | 独自领队 | 沈暮雪第一次没有裴夜在身边，独立带队远征。团队：霍北、姜糖、陈雪、小豆子。她需要独自面对一切决策。**成长线：从"有裴夜兜底"到"自己兜底"。** |
| 124 | 热带废墟 | 跨越凛冬进入南方——极寒消退，灵气复苏后的热带变成了变异丛林。变异植物遮天蔽日，变异昆虫巨大化。全新的环境挑战。 |
| 125 | 丛林法则 | 丛林中的变异兽与北方完全不同——变异蛇、变异蛛、变异蚁群。小豆子的寻宝天赋在丛林中大放异彩。**冒险爽点。** |
| 126 | 域外守卫 | 抵达东南亚遗迹——发现遗迹被域外生物"守卫"着。这些生物比之前遇到的更强——A级。沈暮雪SS级对A级域外生物。**跨级碾压。** |
| 127 | 第二台 | 击杀域外守卫后进入遗迹。第二台装置——与昆仑的类似但构造不同。沈暮雪用灭世极焰关闭它。**第二台关闭。** |
| 128 | 裴夜的信号 | 西欧线传来消息——裴夜那边遇到了麻烦。西欧遗迹的域外守卫是S级。裴夜一人难以同时战斗和关闭装置。**CP牵挂。** |
| 129 | 增援 | 沈暮雪做出决定——让陈雪带队继续去南美，她自己通过空间传送（空间系SS级新能力）前往西欧增援裴夜。**空间传送首次使用。** |
| 130 | 重逢 | 沈暮雪传送到西欧，和裴夜重逢。两人联手——SS级双系火冰 + S级雷——击杀S级域外守卫，关闭第三台装置。**CP并肩名场面。** |

## 段落四：最后两台（第 131–140 章）
**🎯 任务**：南美/非洲最后两台 → 最终决战 → 域外文明真相 → 末世终结 → 大结局

| 章 | 标题 | 情节 / 爽点 |
|----|------|-------------|
| 131 | 倒计时 | 四台已关（昆仑、东南亚、西欧、北美由北美基地关闭）。只剩最后两台——南美和非洲。剩余时间：三个月。其他四台关闭后，这两台的输出暴增十倍。 |
| 132 | 域外之门 | 南美遗迹深处——发现了一个"域外之门"。一个真正的、通往另一个维度的通道。域外生物就是从这里来的。 |
| 133 | 域外文明 | 通道另一侧——一个废墟世界。域外文明也曾存在——但他们被灵气装置摧毁了。他们的世界变成了废墟，灵气装置将两个维度连接在了一起。 |
| 134 | 最终选择 | 沈暮雪面临选择：关闭最后两台装置——会切断两个维度的联系，但也会让域外世界彻底毁灭（它们已经依赖装置的灵气生存）。 |
| 135 | 裴夜的决定 | 裴夜站出来——他愿意进入域外之门，从内部关闭装置。这意味着他可能无法回来。**CP虐点：生死离别。** |
| 136 | 沈暮雪的答案 | 沈暮雪拒绝让裴夜独自冒险。她和他一起——进入域外之门。"你说过的——你需要的我时候，我会在。""我也在。"**双向奔赴巅峰。** |
| 137 | 域外之战 | 两人进入域外世界。面对最后的域外生物群——A级以上成群。沈暮雪灭世极焰 + 裴夜雷系领域——全力以赴。 |
| 138 | 关闭 | 在域外世界的废墟中，找到最后的装置核心。两人联手——灭世极焰+雷系——同时轰击。装置碎裂。两个维度的连接——切断。 |
| 139 | 回归 | 装置关闭的瞬间，域外世界开始崩塌。沈暮雪用空间系——在最后一刻——将两人传送回地球。域外之门永远关闭。 |
| 140 | 新世界 | 所有装置关闭。灵气浓度开始缓慢回归正常水平。变异兽不再进化。天空中的裂缝消失。末世——结束了。沈暮雪站在日出中，裴夜在身边。"结束了吗？""结束了。""那——回家吧。""好。回家。" **全书大结局。** |

## 第四卷必守红线

1. **SS级必须是顶点**——不再继续突破，灭世极焰就是最终技能
2. **裴夜不代劳**——沈暮雪独立关闭至少两台装置，裴夜是辅助不是主角
3. **域外生物要写出"异质感"**——不是更强，而是"不同维度"的可怕
4. **CP分离不超过5章**——Ch122分别→Ch130重逢，中间独立成长
5. **大结局要温暖**——末世结束后的日常，呼应第一卷的"番茄炒蛋"
6. **"回家"作为全书最后一句话**——从重生那天的"末世倒计时"到最后的"回家"，首尾呼应
