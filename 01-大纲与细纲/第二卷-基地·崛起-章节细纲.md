# 第二卷：基地·崛起｜章节细纲

> **卷目标**：兽潮攻城（火系被迫公开，一战封神）→ 基地权斗（孙派系vs沈暮雪团队）→ 陆衍高密度虐渣（多次陷害→反杀→异能被废）→ CP先虐后甜（冷战→和好→确认关系）→ C级变异兽决战 → 突破B级
> **字数预估**：约 30 万字 / 70 章（单章 3300-4000 字）
> **核心节奏**：双线并行（兽潮外患 + 陆衍内忧），两条线交织推进，每 10 章一个高潮

---

## 段落一：暴风前夜（第 51–58 章）
**🎯 任务**：兽潮迫近 + 基地备战 + 陆衍暗中布局 + 姜糖治愈系首秀 + 第一场冲突

| 章 | 标题（暂定） | 情节 / 爽点 |
|----|------|-------------|
| 51 | 兵临城下 | 兽潮前最后一天。基地全员备战。沈暮雪被编入物资调度组，实际上暗藏火系底牌。**钩子：侦察兵回报，C级变异兽正在向城南逼近。** |
| 52 | 各怀心思 | 基地内部的派系暗流。韩志远（军方正统）vs 孙建国（实权派系）。陆衍在孙建国身边出谋划策，暗中盯上了沈暮雪的物资。**引入新角色：孙建国。** |
| 53 | 小豆子的天赋 | 沈暮雪带小豆子（灵狐幼崽）到围墙侦察，小豆子的寻宝天赋首次实战——它感知到了兽潮中有晶核的高等级变异兽位置。**🔥 萌宠+战术价值双重爽点** |
| 54 | 第一声炮响 | 兽潮来了。比预估早了半天。第一批变异犬群冲击城南围墙。沈暮雪在物资调度组掩护下，暗中用火系支援城墙。**战斗开场** |
| 55 | 姜糖的首秀 | 战斗中出现大批伤员，医疗中心人手严重不足。姜糖在沈暮雪保护下，首次使用治愈系异能救人。**🔥 治愈系首秀名场面**。但被孙建国的人看到了。 |
| 56 | 谁在盯着我 | 姜糖使用治愈系的消息被孙建国得知。他派人来"邀请"姜糖加入他的派系。沈暮雪当场拒绝。**冲突起点：孙派系vs沈暮雪** |
| 57 | 城墙告急 | 兽潮第二波，规模远超第一波。城墙南段被C级变异兽撞出了裂缝。守军伤亡惨重。**危机升级** |
| 58 | 不得不出手 | 城墙即将被破，千钧一发。沈暮雪若不出手，基地数万人将陷入灭顶之灾。她做出决定——**暴露火系。** **🔥 卷一超级爆点：D级火系正式公开。段落一高潮收束。** |

## 段落二：一战封神（第 59–68 章）
**🎯 任务**：火系公开大战 + 击退兽潮第一轮 + 一战成名 + 树大招风 + CP第一个虐点

| 章 | 标题 | 情节 / 爽点 |
|----|------|-------------|
| 59 | 火烧城门 | 沈暮雪登上城墙，D级火系全开。火蛇+火箭+压缩火球，一个人守住了一段城墙。**🔥 火系大规模实战，基地数万人亲眼目睹。** |
| 60 | 火系·D+ | 战斗中吸收兽潮晶核，火系突破到D+级。新增能力：火焰领域（小范围温度控制）。**升级爽点** |
| 61 | 裴夜出手 | C级变异兽亲自攻城墙。裴夜以S级雷系正面迎战。一记雷霆将它劈退，但没能杀死——它跑了。**裴夜S级战力再次展示** |
| 62 | 击退 | 兽潮第一轮被击退。基地死伤约两千人，但守住了。沈暮雪成为全基地的焦点人物——**双系异能者（空间+火），D+级。** |
| 63 | 树大招风 | 火系公开后的连锁反应：韩志远想把她编入核心战斗队；孙建国眼红她的实力和物资；各方势力都想拉拢。**权斗线正式开启** |
| 64 | 陆衍的毒计 | 陆衍向孙建国献计：利用沈暮雪的物资合作漏洞，制造"以次充好"的假证据，败坏她的名声。**虐渣铺垫：陆衍的阴招** |
| 65 | 造谣 | 基地里开始流传谣言："沈暮雪的蔬菜有毒""她的药品是过期的""她在物资里掺假"。一些人开始对她指指点点。**虐点：名声被毁** |
| 66 | CP虐点·误解 | 沈暮雪发现裴夜在调查她——她以为裴夜也不信任她。两人爆发冷战。**🔥 CP先虐：沈暮雪的信任危机。"我以为你和她们不一样。"** |
| 67 | 真相 | 其实裴夜是在暗中调查谣言的源头，保护她。真相大白时，沈暮雪愧疚又心疼。**虐后转甜的开始** |
| 68 | 反击开始 | 沈暮雪拿到了谣言源头的证据——指向孙建国的人。她决定反击。**段落二收束，为虐渣高潮蓄力。** |

## 段落三：反击与清算（第 69–80 章）
**🎯 任务**：沈暮雪反击谣言 + 陆衍第一次被清算 + 孙派系受挫 + CP和好确认关系

| 章 | 标题 | 情节 / 爽点 |
|----|------|-------------|
| 69 | 公开对质 | 基地大会上，沈暮雪拿出谣言源头的证据，当众揭露孙建国的人散布谣言。**🔥 打脸名场面** |
| 70 | 陆衍的第一个跟头 | 证据链指向陆衍。他在孙建国面前失了信任，被打入冷宫。**虐渣第一波：陆衍失势** |
| 71 | CP和好 | 裴夜来找沈暮雪。两人深夜在天台谈心，解开误会。裴夜说出那句压了很久的话："我调查你，是因为我怕你受伤。"**🔥 CP确认关系前的关键糖点** |
| 72 | 告白 | 沈暮雪主动。她说："裴夜，我信你。"裴夜沉默了很久，然后说："这一世，我守你。"**🔥 CP正式确认关系。全书最大糖点之一。** |
| 73 | 甜蜜日常 | 确认关系后的日常。裴夜给她送饭、帮她训练、吃霍北的醋（因为霍北总贴着沈暮雪）。**糖点密集章** |
| 74 | 孙建国的不甘 | 孙建国虽然被打了脸，但没死心。他开始谋划更大的动作——联合南域基地的势力。**新威胁伏笔** |
| 75 | 姜糖的成长 | 姜糖的治愈系提升到E级。她开始能治疗更重的伤。同时，她的体质在灵泉水调理下越来越好。**队友成长** |
| 76 | 霍北的突破 | 霍北吸收大量土系晶核，突破到D-级。土系新能力：土墙阵（可同时造多面土墙）。**队友成长** |
| 77 | 小豆子的进化 | 小豆子长到中型犬大小。解锁了第二个天赋：隐匿（短时间消除气息）。头上小角开始生长。**萌宠进化** |
| 78 | 南域来人 | 南域基地派使者来青北"交流"。使者中有人认识陆衍——前世陆衍投靠的正是南域。**虐渣线伏笔升级** |
| 79 | 陆衍的逆袭 | 陆衍通过南域使者重新获得了价值——他知道沈暮雪的底细（空间系、物资量）。南域对他的情报很感兴趣。**陆衍咸鱼翻身** |
| 80 | 暗潮汹涌 | 多方势力汇聚青北。兽潮第二轮即将来临。沈暮雪在备战的同时，还要应对内部权斗。**段落三收束，双线交汇。** |

## 段落四：第二轮兽潮（第 81–92 章）
**🎯 任务**：更大规模兽潮 + C级变异兽再现 + 孙派系暗中作梗 + 陆衍第二次阴谋 + 沈暮雪B级突破

| 章 | 标题 | 情节 / 爽点 |
|----|------|-------------|
| 81 | 第二轮 | 兽潮第二轮爆发。规模是第一轮的三倍。南、东、西三个方向同时进攻。**全面战争** |
| 82 | 三线作战 | 基地兵力分散三段城墙。沈暮雪守南段（主力方向），裴夜游走三段支援。霍北守东段。**团队各显神通** |
| 83 | 内鬼 | 孙建国的人在南段战场故意拖延物资补给，导致南段守军险些崩溃。沈暮雪用空间储备的物资自行补给，化解危机。**🔥 用空间逆转战局** |
| 84 | 陆衍的暗手 | 陆衍趁乱派人潜入沈暮雪的板房，想偷她的物资记录本（里面记录了物资合作的真实数据）。被小银（变异猫）发现并阻止。**虐渣作死** |
| 85 | C级再现 | 那只被裴夜劈跑的C级变异兽回来了，而且变异进化了——它带着一群D级变异兽组成的小队，专门攻击南段城墙的薄弱点。**Boss登场** |
| 86 | 围城 | C级变异兽的战术让南段城墙摇摇欲坠。沈暮雪指挥防御，同时准备亲自迎战C级。**危机** |
| 87 | 决战南墙 | 沈暮雪vs C级变异兽。火系D+级vs C级——差了一个大级别。她被打得很惨，但靠空间系+火系组合勉强撑住。**🔥 Boss战，女主受伤** |
| 88 | 裴夜赶到 | 裴夜击退东段兽潮后赶来支援。两人联手对战C级变异兽。**🔥 CP并肩作战名场面** |
| 89 | 击杀C级 | 裴夜的雷系+沈暮雪的火系联合攻击，终于击杀了C级变异兽。掉落C级晶核。**Boss击杀** |
| 90 | 突破！B级 | 战斗中吸收大量晶核+C级兽核，沈暮雪火系突破到B级。新增能力：炎爆（范围毁灭性攻击）。**🔥 升级大爽点** |
| 91 | 兽潮退去 | C级被杀，兽群溃散。第二轮兽潮被击退。基地惨胜，死伤约五千人。**战后氛围沉重** |
| 92 | 战后清算 | 沈暮雪追查内鬼——孙建国的人拖延补给的事被查实。韩志远借机清洗孙派系。孙建国被削权。陆衍失去靠山，再次陷入困境。**🔥 虐渣第二波：陆衍再次失势。段落四收束。** |

## 段落五：南域阴谋（第 93–102 章）
**🎯 任务**：南域基地的阴谋浮出水面 + 陆衍彻底投靠南域 + 第三次虐渣高潮 + 引出第三卷主线

| 章 | 标题 | 情节 / 爽点 |
|----|------|-------------|
| 93 | 南域的野心 | 南域使者露出真面目——他们来"交流"是假，想吞并青北是真。他们看中了青北的地理位置和防御工事。**新反派势力** |
| 94 | 陆衍的选择 | 孙建国失势后，陆衍彻底投靠南域使者。他出卖了沈暮雪的情报：空间系、物资量、团队构成。**🔥 陆衍彻底黑化** |
| 95 | 暗杀 | 南域派人暗杀韩志远，想制造混乱。暗杀失败，但基地内部人心惶惶。**危机升级** |
| 96 | 真相大白 | 韩志远查出暗杀和南域有关，同时发现陆衍是内鬼。沈暮雪得知陆衍出卖了她的情报。**虐渣高潮铺垫** |
| 97 | 当众清算 | 基地大会上，韩志远当众揭露南域阴谋和陆渐的叛变。沈暮雪站出来，拿出退婚以来的所有证据链。**🔥 虐渣大高潮：陆衍身败名裂** |
| 98 | 废异能 | 陆衍在逃跑中被裴夜截住。沈暮雪亲手用火系废掉了陆衍的异能——"这一世，你连当炮灰的资格都没有了。"**🔥 虐渣终极名场面：异能被废** |
| 99 | 放逐 | 陆衍和林婉婉被驱逐出青北基地。沈暮雪没有杀他——"让你活着看，没有异能、没有靠山的末世，是什么滋味。"**虐渣收尾：比死更狠的惩罚** |
| 100 | 南域宣战 | 南域使者逃回，青北和南域正式对立。第三卷的主线伏笔埋下。**新格局** |
| 101 | 战后重建 | 兽潮和内乱平息，基地开始重建。沈暮雪的团队成为基地核心力量之一。她的名声从"物资商人"变成了"守护者"。**成长里程碑** |
| 102 | 卷尾·星火 | 沈暮雪站在城墙上看日出。裴夜站在她身后。远方的天际线上，一道裂缝隐约可见——第四阶段的征兆。**第二卷收束，双钩子引第三卷。** |

## 段落衔接（第三卷开头）

> 第二卷结束时：沈暮雪B级双系、团队六人全部成长、CP确认关系、陆衍被废放逐、青北vs南域对立、第四阶段裂缝征兆出现。第三卷将进入「极寒凛冬 + 清算沈家 + 裴夜重生身份正式互相坦白 + 南域正面对决」。
