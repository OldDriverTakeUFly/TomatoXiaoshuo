# 第一卷：重生·囤货｜章节细纲

> **卷目标**：重生开局 → 疯狂囤货 → 觉醒异能 → 初步虐渣 → 与男主初遇 → 站稳末世第一步
> **字数预估**：约 20 万字 / 50 章（单章 4000 字左右）
> **核心节奏**：3 章一小爽，10 章一大爽，每 8 章左右一个情节小高潮

---

## 段落一：重生开局（第 1–8 章）
**🎯 任务**：黄金三章抓人 + 金手指亮相 + 立第一个仇敌钩子

| 章 | 标题（暂定） | 情节 / 爽点 |
|----|------|-------------|
| 01 | 死而复生 | 前世末世惨死回忆（被推入尸兽潮）→ 惊醒回到 90 天前。手机日历确认时间。**爽点：金手指——随身空间与灵泉首次显现**。 |
| 02 | 谁在算计我 | 梳理前世记忆：渣男陆衍、毒闺蜜林婉婉、吸血沈家。清点手头资产。**钩子：空间发布第一个升级任务**。 |
| 03 | 第一笔钱 | 去银行取出嫁妆存款 + 股票全部套现，筹到 80 万。空间暴露储物功能，女主欣喜。**小爽：第一次囤货的快感**。 |
| 04 | 疯狂扫货（一） | 网购米面粮油、脱水蔬菜、罐头、压缩饼干，分批收空间。店小二以为她开饭店。**爽点：空间越填越满的满足感**。 |
| 05 | 退婚（上） | 陆衍登门要钱办婚礼，态度傲慢。女主不动声色录音。**铺垫：揭穿其与林婉婉的勾搭**。 |
| 06 | 退婚（中） | 女主跟踪抓到陆衍与林婉婉开房铁证，全程录像拍照。**小爽：猎人反客为主**。 |
| 07 | 退婚（下）| 家宴当众播放视频，退婚 + 索回 50 万彩礼。陆衍身败名裂。**🔥 大爽点：开局第一个打脸名场面**。 |
| 08 | 第一桶金 | 陆衍抵债的房产低价转手套现 120 万。女主冷笑：这只是利息。**段落收束 + 资金升级**。 |

> **段落一钩子**：钱到手了，该扫真正的硬货了。

## 段落二：硬核囤货期（第 9–18 章）
**🎯 任务**：物资囤到极致 + 空间升级 + 灵泉妙用初显 + 邻居线伏笔

| 章 | 标题 | 情节 / 爽点 |
|----|------|-------------|
| 09 | 批发市场扫荡 | 杀入粮油批发市场，整车整车买。商家以为她做救灾供应。**小爽：壕无人性**。 |
| 10 | 药品才是命 | 扫空多家药店：抗生素、止血药、退烧药、避孕药、宠物药。**知识型爽点：前世末世经验带来的精准**。 |
| 11 | 黑市淘装备 | 搞到工兵铲、弩、户外装备、太阳能板、净水器。**新爽源：战斗装备线开启**。 |
| 12 | 灵泉的秘密 | 女主无意中喝了空间灵泉水，疲劳全消、旧疤淡化。**🔥 金手指升级发现**。 |
| 13 | 灵泉催熟 | 用灵泉水浇一颗苹果籽，一夜长成挂果。灵果吃下，体质数值化提升。**空间成长线启动**。 |
| 14 | 小奶猫姜糖 | 邻居女孩姜糖被混混骚扰，女主出手相救。姜糖是隐藏治愈系。**队友线埋伏笔**。 |
| 15 | 囤种子 | 农科院附近收购各类种子、种苗，连果树苗都没放过。**为灵田做铺垫**。 |
| 16 | 那个怪邻居 | 介绍对面神秘的 1601 住户（裴夜，伪装租客观察异常）。两人电梯擦肩。**男主暗线登场**。 |
| 17 | 一场虚惊 | 陆衍被债主追上门来闹事，女主淡定报警 + 反咬一口敲诈勒索。**小爽：用法律反杀**。 |
| 18 | 倒计时三十天 | 看新闻：异常高温、动物异动、政府应急会议。女主冷笑——只有她知道倒计时。**段落收束 + 危机逼近感**。 |

> **段落二钩子**：空间二级，灵田解锁——可以种地了。

## 段落三：天灾降临·觉醒（第 19–28 章）
**🎯 任务**：极热天灾爆发 + 觉醒异能 + 收姜糖 + 与裴夜正式初遇

| 章 | 标题 | 情节 / 爽点 |
|----|------|-------------|
| 19 | 第一天·太阳疯了 | 气温突破 45℃ 飙升，全市停电。众人慌乱，女主空调里吃冰西瓜。**反差爽：末世她过成夏天**。 |
| 20 | 抢购狂潮 | 超市被哄抢一空，水粮价格飞涨。女主淡定看戏，发个朋友圈气人。**小爽：囤货流的终极满足**。 |
| 21 | 第一次救人 | 姜糖中暑晕倒，女主用灵泉水救活，意外发现姜糖治愈系苗子。**队友绑定**。 |
| 22 | 觉醒！空间系 | 灵气复苏第一波，女主在睡梦中觉醒空间异能（外界可见的），公开身份——空间系。**🔥 大爽点：稀有异能惊艳全场**。 |
| 23 | 隐藏的火 | 当夜她发现还有第二异能：火系。决定隐藏，只露空间系。**伏笔：火系是后期杀招**。 |
| 24 | 物资换人情 | 用空间存的常温奶和药品，与楼内邻居换取信息与人情。**末世初期人脉积累**。 |
| 25 | 变异鼠群 | 小区地下车库出现第一批变异动物——变异鼠。众人恐慌。**第一场小战斗**。 |
| 26 | 一铲之威 | 女主用工兵铲 + 隐藏火系，单杀变异鼠头领救下整栋楼。**🔥 战斗初爽**。 |
| 27 | 他叫裴夜 | 神秘邻居裴夜出手相助清理鼠群，雷系异能惊鸿一现。两人第一次正式对话。**🔥 男主登场名场面**。 |
| 28 | 同行一程 | 裴夜目光落在女主工兵铲的灼烧痕迹上，若有所思。两人结伴下楼取物资，话不多但默契。**CP 暗线启动**。 |

> **段落三钩子**：极热刚适应，异变却不止于此——变异兽要来了。

## 段落四：异兽来袭·立威（第 29–38 章）

| 38 | 一战成名 | 兽潮退去，女主成为小区核心人物。她立规矩：能者多劳，不养闲人。**立威 + 势力雏形**。 |
