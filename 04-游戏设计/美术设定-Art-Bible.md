## 一、风格定调

### 1.1 风格定义

**精细日系像素 + 氛围光影**

不是怀旧粗糙像素——而是每颗像素都有意义、用最少的格子传递最多信息的现代像素风。

核心原则：

| 原则 | 说明 |
|------|------|
| 像素有意义 | 每一颗像素都承担色彩/轮廓/光影的功能，不浪费 |
| 轮廓清晰 | 角色在任何背景下都能一眼识别（深色轮廓线+高对比度） |
| 光影氛围 | 用渐变和明暗塑造空间感，而非纯平面填色 |
| 动画流畅 | 关键帧之间用插值/挤压拉伸传递动感 |

### 1.2 参考游戏取材

| 参考游戏 | 取什么 | 不取什么 |
|----------|--------|----------|
| 《星露谷物语》 | 种田系统的温馨感、角色辨识度 | 过于明亮的卡通配色 |
| 《死亡细胞》 | 战斗动画的流畅度、特效冲击力 | 暗黑哥特氛围 |
| 《八方旅人》 | 场景的景深光影效果 | HD-2D的复杂3D背景 |
| 《泰拉瑞亚》 | 像素粒子特效的质感 | — |

### 1.3 视觉关键词

`末世` `废墟` `温暖（空间内）` `危险（空间外）` `异能光效` `反差感`

## 三、角色设计

### 3.1 像素布局通用规则

32×32画布中，角色按以下比例排布：

```
32 ┌──────────────────────────┐
   │                          │
   │     头部区 (头部+头发)    │  ← Y: 0-10 (11行)
   │                          │
   │     面部区 (五官)         │  ← Y: 6-10 (面部细节)
   │                          │
   │     身体区 (躯干+手臂)    │  ← Y: 11-24 (14行)
   │                          │
   │     腿部区 (双腿)         │  ← Y: 25-31 (7行)
   │                          │
32 └──────────────────────────┘
   X:←           →           →
      8列          16列        8列
    (左侧)      (中心主体)   (右侧)
```

**关键约束**：
- 面部最低2×3像素用于眼睛（深色），保证表情可读
- 轮廓线统一使用比主色深2级的同色系，不使用纯黑
- 阴影统一在右下方（光源左上方），1像素偏移

### 5.3 六大场景概念

#### 场景一：灵泉空间

**氛围**：温暖的避风港。末世沙漠中的绿洲。

关键元素：
- 灵泉眼：中央圆形水池，蓝色发光水面，4帧水波纹动画
- 灵田：黑土垄，作物根据成长阶段有4种外观（种子/幼苗/生长/成熟）
- 灵果树木：小果树，果实有发光效果
- 仓库：木质货架，堆放物资袋
- 光照：暖黄光从上方洒下，水面反射光斑

Tile需求：土墙、黑土垄、水面（4帧）、灵泉边缘石、木地板、木货架、草丛、果实

#### 场景二：极热天灾城市

**氛围**：被太阳烤焦的废墟。

关键元素：
- 开裂柏油路面（裂缝中有热气冒出——2帧蒸汽动画）
- 干枯行道树（枝干扭曲，无叶）
- 废弃车辆（铁锈色，玻璃碎）
- 倾斜电线杆（电线下垂）
- 远景：扭曲的高楼天际线（热浪变形效果——远景层加正弦扰动）
- 灯光层：橙红色叠加，模拟高温

Tile需求：柏油路面（3种裂纹变体）、人行道、废车、枯树、电线杆、铁丝网

#### 场景三：青北基地

**氛围**：秩序井然的人类据点。

关键元素：
- 混凝土围墙（顶部铁丝网）
- 铁皮板房（各种颜色——蓝、绿、灰）
- 国旗杆（旗帜4帧飘动动画）
- 沙袋工事
- 探照灯（光锥效果——灯光层）
- 哨塔

Tile需求：混凝土墙、铁皮板房×3色、沙袋、铁丝网、国旗、探照灯光锥

#### 场景四：极寒冰原

**氛围**：白茫茫的死亡世界。

关键元素：
- 积雪地面（表面冰晶高光）
- 冰裂缝（深蓝渐变向下消失）
- 冻结的建筑残骸（冰包裹的汽车/建筑）
- 暴风雪粒子（前景层，白点横向飘移）
- 远景：冰山轮廓
- 灯光层：蓝白色叠加，边缘暗角

Tile需求：积雪（3种纹理变体）、冰面、冰裂缝、冻废墟、枯冰树

#### 场景五：昆仑遗迹

**氛围**：远古文明的庄严废墟。

关键元素：
- 不明金属墙壁（暗金色，有符文刻纹——符文微弱发光）

- 灵气粒子（漂浮的金色光点）
- 结界光膜（半透明蓝光）
- 灯光层：紫色+金色交替

Tile需求：金属墙（3种符文变体）、壁画×4、装置底座、结界光膜、灵气粒子

#### 场景六：域外世界

**氛围**：不属于地球的异维度空间。

关键元素：
- 虚空背景（纯黑+紫色星云——2帧呼吸效果）
- 漂浮的碎片（建筑残骸、石块悬浮——3帧上下浮动）

### 7.3 字体方案

中文像素字体是难点——推荐方案：

| 用途 | 字体 | 说明 |
|------|------|------|
| 正文/对话 | 站酷快乐体 / 罗德岛字体（像素版） | 圆润清晰，可读性好 |
| 标题/数字 | Pixelify Sans / Press Start 2P（英文部分） | 经典像素感 |
| 备选 | 自制点阵字体（12×12汉字） | 最纯正但工作量极大 |

> **推荐**：使用矢量字体缩小渲染 + pixel-perfect后处理，而非严格像素字体。在移动端12px汉字用真像素字体可读性极差。
