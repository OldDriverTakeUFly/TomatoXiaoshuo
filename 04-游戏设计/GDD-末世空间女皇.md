# 游戏设计文档（GDD）
# 《末世空间：女皇》

> **版本**：v1.0 | **日期**：2026-07-14 | **状态**：概念设计
> **平台**：移动端（iOS/Android） | **引擎**：Unity 2D / Godot
> **团队**：1-3人独立团队 | **美术**：像素风
> **开发周期**：MVP 6个月 / 正式版 12个月

## 一、游戏概述

### 1.1 一句话简介

> 一款末世生存×种田经营的像素风RPG——在毁灭的世界里，用你的灵泉空间种地、打怪、建基地，从孤身一人成长为空间女皇。

### 1.2 核心体验（一句话）

**"出门打怪收集物资 → 回空间种田升级 → 强化后打更强的怪"** 的三角循环。

### 1.3 类型标签

`末世生存` `种田经营` `RPG` `像素风` `异能战斗` `宠物系统`

### 1.4 目标受众

| 维度 | 描述 |
|------|------|
| 年龄 | 18-35岁 |
| 性别 | 女性为主（60%），男性（40%） |
| 画像 | 番茄/起点末世文读者、星露谷/生存类游戏玩家、放置经营类玩家 |
| 参考竞品 | 《星露谷物语》《辐射避难所》《幻兽帕鲁》《明日方舟》 |

### 1.5 商业化模式

免费下载 + 内购

| 内购项 | 价格 | 说明 |
|--------|------|------|
| 月卡 | ¥18/月 | 每日领取灵泉水×10、灵果×1 |
| 空间扩展包 | ¥6-30 | 扩大空间储物面积/灵田面积 |
| 角色皮肤 | ¥6-12 | 像素风角色换装（沈暮雪/裴夜/姜糖等） |
| 小豆子皮肤 | ¥6 | 灵狐幼崽的不同毛色/外观 |
| 去广告 | ¥12 | 一次性，永久移除激励视频以外的广告 |
| 激励视频 | 免费 | 看广告获得双倍掉落/灵泉加速等 |

> **设计原则**：不卖数值。内购只卖"便利"和"外观"，不卖战斗力。保证零氪玩家也能通关。

## 二、核心玩法循环

### 2.1 三角循环图

```
        ┌─────────────────────────┐
        │    灵泉空间（经营层）     │
        │                         │
        │  种灵果 → 吃灵果 → 永久属性↑
        │  种蔬菜 → 交基地 → 声望/资源↑
        │  泡灵泉 → 恢复体力/灵气
        │  管理仓库 → 物资分类整理
        │                         │
        └────────┬────────────────┘
                 │ 属性/等级提升
                 ▼
    ┌─────────────────────────────┐
    │      荒野探索（战斗层）       │
    │                             │
    │  选择关卡 → 进入荒野地图      │
    │  小豆子寻宝 → 定位晶核/宝箱   │
    │  遭遇变异兽 → 异能战斗        │
    │  掉落晶核/材料/种子           │
    │  体力归零/背包满 → 返回       │
    │                             │
    └────────────┬────────────────┘
                 │ 晶核/材料/种子带回
                 ▼
    ┌─────────────────────────────┐
    │     基地互动（社交层）        │
    │                             │
    │  交物资 → 提升基地声望        │
    │  声望↑ → 解锁队友/功能/商店   │
    │  招募队友 → 编入探索队伍      │
    │  队友升级 → 解锁合击/被动     │
    │  剧情推进 → 解锁新关卡区域    │
    │                             │
    └─────────────────────────────┘
```

### 2.2 单次Session设计（5-15分钟）

```
登录（30秒）
  → 收灵田作物、泡灵泉（2分钟）
  → 出城探索1-3个关卡（5-10分钟）
  → 回空间种地、升级、交物资（2分钟）
  → 推进剧情/做支线（可选，3分钟）
退出
```

> **设计原则**：单次Session不超过15分钟。碎片化友好。但有深度内容供长时间玩家持续游玩。

## 三、核心系统详细设计

### 3.1 灵泉空间（经营层）

这是游戏的**核心差异化系统**——也是小说金手指的直接还原。

#### 3.1.1 空间布局

```
┌─────────────────────────────────────┐
│            灵泉空间                  │
│                                     │
│  ┌─────────┐  ┌──────────────────┐ │
│  │ 灵泉眼   │  │     灵田区       │ │
│  │ (温泉池) │  │  ┌──┐┌──┐┌──┐  │ │
│  │  恢复    │  │  │番││白││黄│  │ │
│  │  修炼    │  │  │茄││菜││瓜│  │ │
│  │          │  │  └──┘└──┘└──┘  │ │
│  └─────────┘  │  ┌──┐┌──┐┌──┐  │ │
│               │  │水││玉││辣│  │ │
│  ┌─────────┐  │  │稻││米││椒│  │ │
│  │ 储物仓库 │  │  └──┘└──┘└──┘  │ │
│  │ (无限大  │  └──────────────────┘ │
│  │  但分类) │                       │
│  └─────────┘  ┌──────────────────┐ │
│               │    灵果园区       │ │
│               │  🌳🍎 🌳🍇 🌳🥝  │ │
│               └──────────────────┘ │
└─────────────────────────────────────┘
```

#### 3.1.2 灵泉眼

| 功能 | 说明 | 数值 |
|------|------|------|
| 恢复体力 | 泡温泉恢复外出消耗的体力 | 每分钟恢复10点，上限100 |
| 恢复灵气 | 泡温泉恢复战斗灵气值 | 每分钟恢复5%，上限100% |
| 修炼加速 | 泡温泉时异能经验获取×2 | 需消耗灵泉水（道具） |
| 灵气过载 | 泡超过10分钟会"过载"，降低当天效率 | 防止无限挂机 |

> **设计意图**：灵泉是"体力/灵气恢复点"也是"修炼加速器"，但有过载限制防止无限挂机。

#### 3.1.3 灵田系统

**种植机制**：
- 每块田（3×3格子）可以种一种作物
- 作物有生长周期（空间内时间，约真实时间10分钟-2小时）
- 灵泉水浇灌可缩短30%生长时间
- 收获的作物分为三类用途

| 作物类型 | 示例 | 用途 |
|----------|------|------|
| 蔬菜 | 番茄、白菜、黄瓜 | 交基地声望 / 烹饪料理（临时buff） |
| 粮食 | 水稻、玉米、小麦 | 基地物资（战略资源） |
| 灵果 | 苹果、葡萄、奇异果 | **吃掉 → 永久属性提升**（核心成长） |

**灵果系统（重点）**：

灵果是玩家角色永久属性提升的主要来源——直接还原小说设定。

| 灵果类型 | 属性提升 | 生长周期 | 备注 |
|----------|----------|----------|------|
| 普通灵果 | 体质+1 | 10分钟 | 常规产出 |
| 精良灵果 | 体质+2 / 力量+1 | 30分钟 | 需要中级种子 |
| 稀有灵果 | 全属性+1 | 1小时 | 需要高级种子 |
| 传说灵果 | 全属性+2 + 特殊效果 | 2小时 | 需要Boss掉落种子 |

> **设计意图**：灵果让"种田"不只是赚钱手段，而是**角色成长的直接途径**。玩家会真心在乎自己的田——因为每颗灵果都是实打实的属性。

#### 3.1.4 空间升级

空间随等级解锁，需要消耗"空间晶核"（特殊任务/成就获得）。

| 空间等级 | 解锁内容 | 所需条件 |
|----------|----------|----------|
| Lv1 | 储物100格 + 灵泉 + 灵田3块 | 初始 |
| Lv2 | 储物300格 + 灵田6块 + 灵果园 | 主线Ch3 |
| Lv3 | 储物500格 + 灵田9块 + 时间加速修炼室 | 主线Ch8 |
| Lv4 | 储物∞ + 灵田12块 + 灵兽栖息地 | 主线Ch15 |
| Lv5 | 自定义布局 + 独立生态系统 | 主线Ch30 |

### 3.2 异能战斗系统（战斗层）

#### 3.2.1 战斗模式

**横屏即时动作制**——玩家手动操作沈暮雪，在2D横版地图中移动、闪避、释放异能。

**操作方式（手游适配）**：

```
┌─────────────────────────────────────────────┐
│                                             │
│              [战斗画面]                      │
│                                             │
│    沈暮雪 ←→ 移动（虚拟摇杆）               │
│                                             │
│    变异兽                                    │
│                                             │
├─────────────────────────────────────────────┤
│  [摇杆]  [火系] [冰系] [极焰] [道具] [逃跑]  │
│         技能1  技能2  大招                   │
└─────────────────────────────────────────────┘
```

- **左下虚拟摇杆**：移动
- **右下技能按钮**：4个技能位 + 1个道具位
- **上滑/下滑**：闪避（前冲/后跳）

#### 3.2.2 异能技能体系

玩家拥有三种异能模式，**战斗中可以随时切换**：

**火系（输出型）**

| 技能 | 效果 | 灵气消耗 | 解锁等级 |
|------|------|----------|----------|
| 火球 | 远程单体伤害 | 5 | 初始 |
| 火蛇 | 直线穿透伤害 | 15 | F级 |
| 火焰领域 | 范围持续灼烧 | 30 | D级 |
| 压缩火球 | 高伤单体+溅射 | 25 | D+级 |
| 炎爆 | 范围爆炸 | 50 | C级 |
| 炎帝领域 | 大范围高温领域 | 80 | B级 |

**冰系（控制型）**

| 技能 | 效果 | 灵气消耗 | 解锁等级 |
|------|------|----------|----------|
| 冰锥 | 远程单体+减速 | 5 | B级觉醒 |
| 冰墙 | 创造掩体 | 15 | B级 |
| 冰封 | 定身单体3秒 | 25 | B级 |
| 寒冰领域 | 范围减速+持续冰伤 | 40 | A级 |

**极焰（融合大招型）**

| 技能 | 效果 | 灵气消耗 | 解锁条件 |
|------|------|----------|----------|
| 极焰弹 | 火冰融合弹，无视护甲 | 50（火+冰各25） | 冰系觉醒后 |
| 极焰领域 | 热胀冷缩碎甲领域 | 100 | A级 |
| 灭世极焰 | 全屏毁灭性攻击 | 全部灵气 | SS级 |

> **极焰特殊机制**：使用极焰需要火系和冰系灵气各消耗一半——这意味着玩家需要同时管理两种灵气。火系和冰系各有独立的灵气条。

**灵气条设计**：

```
HP: ████████████████████ 100/100
火灵气: ████████░░░░░░░░░░░░ 40/100
冰灵气: ████████████████████ 100/100
极焰槽: ░░░░░░░░░░░░░░░░░░░░ 0/100 （战斗中逐渐积累）
```

极焰槽在战斗中通过火/冰交替使用来积累——每切换一次异能+10极焰槽。积满100后可以释放极焰大招。

> **设计意图**：鼓励玩家频繁切换异能——而不是只用火系输出。交替使用 = 更高的DPS（因为极焰大招）。还原小说中"冰火融合"的核心设定。

#### 3.2.3 属性系统

| 属性 | 影响 | 提升方式 |
|------|------|----------|
| 体质 | HP上限、抗性 | 吃灵果、泡灵泉修炼 |
| 力量 | 物理攻击、负重 | 吃灵果、装备 |
| 灵力 | 异能攻击、灵气上限 | 吃灵果、吸收晶核、升级 |
| 感知 | 暴击率、闪避率、视野 | 吃灵果、装备 |
| 速度 | 移动速度、攻击速度 | 吃灵果、装备 |

每项属性0-100。灵果可以提升任意一项——但越往后需要的灵果越多（递减机制）。

#### 3.2.4 等级体系

直接使用小说的异能等级：

| 等级 | 称呼 | 所需经验 | 解锁内容 |
|------|------|----------|----------|
| F | 觉醒期 | 0 | 火球 |
| E | 稳定期 | 500 | 火蛇 |
| D | 成长期 | 2000 | 火焰领域 |
| D+ | 成长期巅峰 | 5000 | 压缩火球 |
| C | 精英期 | 15000 | 炎爆 |
| B | 强者 | 50000 | 炎帝领域 |
| A | 王阶 | 200000 | 冰系觉醒+极焰 |
| SS | 帝阶 | 1000000 | 灭世极焰 |

> 经验来源：击杀变异兽（主）、吸收晶核（辅）、泡灵泉修炼（辅）。

### 3.3 探索系统

#### 3.3.1 关卡结构

**关卡制**——而非开放世界。每个关卡是一个独立的2D横版地图。

关卡类型：

| 类型 | 说明 | 时长 | 示例 |
|------|------|------|------|
| 普通关卡 | 清怪+收集 | 3-5分钟 | 城南荒野、高速废墟 |
| Boss关卡 | 单挑强敌 | 5-8分钟 | 变异鼠王、变异虎、C级变异兽 |
| 兽潮防御 | 守城波次战 | 5-10分钟 | 南段城墙防御战 |
| 剧情关卡 | 纯剧情+少量战斗 | 3-5分钟 | 退婚打脸、裴夜初遇 |
| 特殊关卡 | 解谜/生存/护送 | 5分钟 | 遗迹探索、域外遭遇 |

#### 3.3.2 关卡区域解锁（对应小说进度）

| 区域 | 对应剧情 | 解锁条件 | 主要敌人 |
|------|----------|----------|----------|
| 青城南区 | 第1卷段落一 | 初始 | 小混混（教学） |
| 青城批发市场 | 第1卷段落二 | Ch3后 | — (收集关) |
| 青城地下车库 | 第1卷段落三 | Ch15后 | 变异鼠F-E级 |
| 青城小区 | 第1卷段落四 | Ch20后 | 变异犬D级 |
| 北上公路 | 第1卷段落五 | Ch30后 | 变异兽+流民 |
| 青北基地外围 | 第1卷段落六 | Ch35后 | 变异犬群 |
| 青北基地·兽潮 | 第2卷段落一 | Ch40后 | 变异虎C级+兽潮 |
| 城西荒野 | 第2卷自由探索 | Ch45后 | D级变异兽群 |
| 极寒冰原 | 第3卷 | Ch55后 | 冰系变异兽 |
| 昆仑遗迹 | 第4卷 | Ch65后 | 域外生物 |
| 域外世界 | 终章 | Ch70后 | 域外Boss |

#### 3.3.3 小豆子寻宝系统

**战斗中**：小豆子在地图上用爪子指方向，标记晶核/宝箱/隐藏敌人位置。

**探索中**：小豆子的感知范围随等级提升，可以发现隐藏路径和稀有材料。

| 小豆子等级 | 感知范围 | 额外能力 |
|------------|----------|----------|
| 幼体 | 5格 | 寻宝标记 |
| 成长期 | 10格 | 隐匿（短暂消失避开敌人） |
| 成熟期 | 全图 | 寒霜吐息（辅助攻击） |
| 完全体 | 全图+穿墙 | 参战（独立攻击） |

> **设计意图**：小豆子不只是宠物——它是探索系统的核心。没有它，玩家找不到隐藏的晶核和宝箱。这让宠物系统有实际的游戏价值。

### 3.4 基地系统（社交层）

#### 3.4.1 青北基地

基地是你的"大本营"——主线剧情在这里推进，队友在这里招募。

**基地声望**：通过供应物资提升。

| 声望等级 | 所需物资 | 解锁内容 |
|----------|----------|----------|
| 陌生 | — | 基本商店 |
| 熟悉 | 100蔬菜+50粮食 | 霍北加入 |
| 信赖 | 500蔬菜+200粮食+50药品 | 姜糖加入 |
| 核心 | 2000蔬菜+1000粮食+200药品+10晶核 | 陈雪加入 |
| 守护者 | 持续供应 | 空间女皇称号+特殊功能 |

#### 3.4.2 队友系统

| 队友 | 加入条件 | 战斗定位 | 特殊技能 |
|------|----------|----------|----------|
| 霍北 | 基地声望"熟悉" | 近战坦克 | 土墙（造掩体）、嘲讽 |
| 姜糖 | 基地声望"信赖" | 治疗 | 治愈之手（恢复HP）、群体回血 |
| 陈雪 | 基地声望"核心" | 中距离输出 | 风刃、火风刃（合击） |
| 裴夜 | 主线Ch45（剧情固定加入） | 全能型（不可操控） | 战斗中自动支援，不操控 |

> **裴夜特殊设计**：裴夜不作为可操控角色——他在剧情关键时刻自动出现支援（对应小说中"你需要的我时候我会在"）。这样既还原了小说人设，又避免了"S级角色破坏游戏平衡"的问题。

**合击系统**：特定队友组合解锁合击技。

| 组合 | 合击技 | 效果 |
|------|--------|------|
| 沈暮雪+陈雪 | 炎帝风刃 | 火属性风刃，范围灼烧 |
| 沈暮雪+霍北 | 熔岩壁垒 | 火系强化土墙，反击伤害 |
| 沈暮雪+姜糖 | 灵泉复苏 | 全队回血+灵气恢复 |
| 全队齐出 | 女皇降临 | 全屏AOE+全队buff |

### 3.5 烹饪与料理系统

直接还原小说中"番茄炒蛋"的名场面。

**料理效果**：食用后获得临时buff（持续N场战斗）。

| 料理 | 所需食材 | 效果 |
|------|----------|------|
| 番茄炒蛋 | 番茄×2+鸡蛋×1 | 火系伤害+10%，持续3场 |
| 白菜炖粉条 | 白菜×2+粉条×1 | 体力上限+20，持续3场 |
| 腊肉炒蒜苗 | 腊肉×1+蒜苗×2 | 力量+5，持续3场 |
| 灵果沙拉 | 灵果×3 | 全属性+3，持续5场 |
| 火锅 | 肉×2+菜×3+调料×1 | 全队回满HP+灵气 |

## 五、数值框架（核心）

### 5.1 伤害公式

```
最终伤害 = (基础攻击 × 异能倍率 × 属性加成 × 等级系数) × (1 - 敌人抗性)
```

- **基础攻击**：由灵力属性决定
- **异能倍率**：每个技能固定倍率（如火球=1.5x，炎爆=5x）
- **属性加成**：(灵力/10) —— 每10点灵力+10%伤害
- **等级系数**：(1 + 等级差 × 0.1) —— 高等级打低等级有优势
- **敌人抗性**：每种变异兽对不同元素有抗性（冰兽抗冰，弱火）

### 5.2 成长曲线

```
属性提升所需灵果数量：
1-10点：   每点1颗普通灵果
11-30点：  每点2颗普通灵果（或1颗精良）
31-60点：  每点3颗精良灵果（或1颗稀有）
61-100点： 每点2颗稀有灵果（或1颗传说）
```

> **设计意图**：前期成长快（爽），后期成长慢但有意义（肝）。和小说的成长节奏一致。

### 5.3 关卡难度曲线

| 关卡序号 | 推荐等级 | 敌人等级 | Boss |
|----------|----------|----------|------|
| 1-5 | F | F-E | 小混混头目 |
| 6-10 | E | E-D | 变异鼠王 |
| 11-15 | D | D-D+ | 变异犬首领 |

| 21-30 | C | C-B+ | 变异虎+变异鹰+变异蝎 |
| 31-40 | B | B | B级冰系变异兽 |

## 十、附录：小说IP使用建议

### 10.1 如果小说已在番茄签约

番茄小说签约作品的游戏改编权通常归平台或作者所有——需要确认：
- 签约合同中是否包含"游戏改编权"条款
- 如果平台拥有——需要向平台申请授权
- 如果作者保留——作者本人可以授权

### 10.2 如果不使用小说IP

可以保留游戏玩法设计，修改角色名称和剧情：
- 沈暮雪 → 自定义主角
- 灵泉空间 → 随身空间（通用设定）
- 异能体系 → 通用末世异能设定

玩法本身（种田+战斗+经营）不依赖IP——可以独立成立。
