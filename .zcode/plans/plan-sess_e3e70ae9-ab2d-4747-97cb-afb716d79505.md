# 美术设定文档（Art Bible）计划

## 参数确认
- 像素精度：32×32（角色），32×32（场景tile）
- 文档深度：全套（风格定调 + 色板 + 全品类设计）

## 文档结构

### 一、风格定调
- 像素风格定义（精细日系像素 + 光影氛围）
- 参考游戏视觉分析（星露谷/死亡细胞/八方旅人各取什么）
- 整体视觉关键词

### 二、色彩系统
- 6大场景类型各一套色板（灵泉空间/极热天灾/基地/极寒/域外遗迹/域外世界）
- 角色通用色板（肤色/发色/服装/特效色）
- 特效色板（火系/冰系/极焰/雷系/治愈）

### 三、角色设计（5个主角 + 3个敌人范例）
每个角色含：
- 像素布局描述（32×32网格上的位置/比例/关键像素分布）
- 配色方案（具体色值）
- 动画需求清单（idle/walk/attack/skill/hurt等各几帧）
- ASCII参考图（用字符画出像素布局示意）
- 角色涉及：沈暮雪 / 裴夜 / 姜糖 / 霍北 / 陈雪 / 小豆子

### 四、敌人设计（变异兽体系）
- F级~A级变异兽的视觉进化规则（体型/颜色/细节递增）
- 3个范例：变异鼠(F) / 变异虎(C) / 域外生物(A)
- Boss设计原则（辨识度/弱点视觉提示/体型比例）

### 五、场景设计
- tile规格（32×32，场景图层拆分：前景/中景/远景/灯光层）
- 6大场景的概念描述+色彩氛围+关键元素清单
- 场景物件清单（可交互物件/装饰物件/危险物件）

### 六、特效设计
- 异能特效（火/冰/极焰/雷系/治愈，每系3-4帧逐帧描述）
- 环境特效（天气/破坏/拾取）
- UI特效（升级/解锁/提示）

### 七、UI设计
- 界面布局（竖屏主界面/横屏战斗/灵泉空间/基地/菜单）
- 图标系统（物资/技能/队友/状态 buff 的像素图标规格）
- 字体推荐（中文像素字体方案）

### 八、动画规格表
- 全部角色的动画清单+帧数+尺寸
- 场景动画（灵泉涌动/作物生长/天气循环）

### 九、美术资源清单（Budget）
- 角色/敌人/场景/特效/UI各需多少张图
- 总像素画工作量估算（张数×帧数）
- Aseprite文件组织结构建议

## 输出
- 一个文件：`04-游戏设计/美术设定-Art-Bible.md`
- 内含ASCII参考图、色值表、帧数表等可直接交给美术执行的规格
