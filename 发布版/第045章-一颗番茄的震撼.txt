第四十五章 一颗番茄的震撼

入城第三天。

沈暮雪发现了一个问题——基地的伙食，很难吃。

C 区的食堂每天供应两顿饭，早上一顿，傍晚一顿。

菜品固定：压缩饼干、方便面（没有调料包）、偶尔有一碗白粥配咸菜。

量少，味道差，营养单一。

大部分幸存者已经末世一个多月没吃过正经饭了，能有口吃的就不错——没人抱怨。

但沈暮雪的团队不一样。

他们跟了沈暮雪一个多月，嘴被养刁了。

在青城的时候，沈暮雪隔三差五就用灵田食材给他们做饭——灵泉水浇灌的番茄、白菜、黄瓜、米饭，每一样都新鲜、美味、营养丰富。

现在让他们啃压缩饼干？

"雪姐……"姜糖啃了一口压缩饼干，整张脸皱成了包子，"这个好难吃……"

"忍忍。"沈暮雪自己也啃了一口。

确实难吃。

干巴巴的，像在嚼锯末。

霍北更直接——他咬了一口压缩饼干，吐了。

"沈妹子，这玩意儿是人吃的？"

"末世里有的吃就不错了。"张姐苦笑。

小周默默地啃着，没说话——但眉头也皱得紧紧的。

沈暮雪看着团队的表情，心里盘算了一下。

她有空间——里面有源源不断的新鲜蔬菜、粮食、水果、肉类。

但基地有物资申报制度。

如果她突然拿出大量新鲜食物，会引起注意。

——但如果只是给团队改善伙食，量不大，就不算太扎眼。

她做了一个决定。

"今晚我来做饭。"

"真的？！"姜糖眼睛一亮。

"嗯。"

消息传开，团队的士气瞬间提升了一个档次。

霍北甚至主动跑去食堂借锅碗瓢盆——食堂的管理员看他是个壮汉，又听说是"C 区新来的那批人要自己开伙"，也没为难，借了一套厨具。

当天傍晚，沈暮雪在板房门前的空地上，支起了一口铁锅。

她从空间里——从一个看起来很普通的帆布袋里（做给旁人看的）——取出了食材。

番茄——灵田种的，红艳艳的，每一个都比拳头大，表皮光滑得像打了蜡。

鸡蛋——空间里养的鸡下的（是的，她在灵田旁边养了几只鸡，灵泉水喂养，下的蛋蛋黄都是橙红色的）。

大米——灵田种的，晶莹剔透，散发着一股天然的米香。

葱、蒜、盐、糖——一应俱全。

她开始做饭。

淘米、煮饭。

番茄切块，鸡蛋打散。

热锅，倒油——她甚至带了猪油。

"嗞——"

蛋液下锅的瞬间，一股浓郁的蛋香弥漫开来。

周围的空气突然安静了。

不是真的安静——是人声变低了。

C 区的幸存者们，一个接一个地停下了手里的动作，鼻子不自觉地抽动。

那是——

鸡蛋的味道。

真正的、新鲜的、热腾腾的鸡蛋的味道。

末世四十多天了，大部分人已经忘了这个味道长什么样。

"什么味儿？"有人凑过来。

"有人……在做饭？"

"不是压缩饼干的味道……这是……炒菜？！"

人越聚越多。

沈暮雪不为所动，专注于锅里。

番茄下锅。

"嗞啦——"

番茄的汁水在高温下迸溅，酸甜的香气混合着蛋香，形成了一种让人瞬间饿到胃抽筋的——

人间烟火气。

她翻炒了几下，加了盐和一点点糖提鲜——

出锅。

一盘红黄相间、汁水丰盈、散发着诱人香气的番茄炒蛋，摆在桌上。

旁边是一锅冒着热气的白米饭——米粒饱满莹白，香气扑鼻。

姜糖第一个凑过来，看着那盘番茄炒蛋，眼睛亮得像星星。

"雪姐！！"

"开饭。"沈暮雪盛了一碗饭，浇了一大勺番茄炒蛋的汤汁。

姜糖接过碗，扒了一口饭——

她愣住了。

然后——

眼泪"唰"地流了下来。

"怎么了？"沈暮雪一惊。

"好好吃……"姜糖哽咽着，一边哭一边往嘴里塞饭，"太好吃了……呜呜呜……"

她哭得稀里哗啦，但嘴一直没停。

霍北更夸张——他一口气吃了三碗饭，盘底的汤汁都用米饭蘸干净了。

"沈妹子！"他拍了拍肚皮，一脸满足，"你这手艺，放末世前能开饭店！"

张姐吃得斯文一些，但眼眶也红了。

小周沉默地把盘子里的最后一块番茄夹起来，放进嘴里，细细地嚼了很久。

小刘的女儿——那个五岁的小姑娘——捧着碗，笑得眉眼弯弯，小声说了句："阿姨做的饭，比妈妈做的还好吃。"

小刘在旁边红了脸："这孩子……"

就连宋老，都多吃了一碗。

他放下筷子，看着沈暮雪，目光复杂。

"丫头，你这食材——不普通。"

沈暮雪心里微微一动。

宋老是个明白人。

"好东西。"她没有否认，也没有细说，"以后跟着我，不会让大家饿着。"

宋老看着她，点了点头。

他没追问。

但他的眼神里，多了几分郑重。


这顿饭的动静闹得不小。

C 区的幸存者们都看见了——那个新来的年轻女人，在板房门口做了一锅番茄炒蛋和白米饭。

新鲜的番茄。

新鲜的鸡蛋。

新鲜的大米。

在压缩饼干和方便面撑起了整个基地伙食的当下——

这简直是暴殄天物……不，是奢侈。

是想象不到的奢侈。

"她哪来的新鲜蔬菜？"

"不知道，说是自己囤的。"

"末世前囤的？一个多月了还能保鲜？"

"人家有空间系异能啊——你忘了？登记的时候我看见了，空间系。"

"空间系能保鲜？"

"当然能。空间系就是人形仓库，放进去的东西不会坏。"

窃窃私语在 C 区传开了。

到第二天，不只是 C 区——连隔壁的 B 区和 D 区，都有人听说了。

"C 区新来了一个空间系女异能者，有大量新鲜食材，还做了一顿番茄炒蛋。"

消息像涟漪一样扩散。

沈暮雪没有刻意宣传，但她也没有刻意低调。

因为她知道——

在末世里，食物就是最好的名片。

一盘番茄炒蛋，比任何自我介绍都管用。

它传递了三个信息：

第一，她有物资。大量的、新鲜的、高品质的物资。

第二，她有空间系异能。可以长期保存物资。

第三，她愿意分享。至少对团队内部，她不吝啬。

这三个信息加在一起——

足以让她在青北基地，迅速建立起"有价值的人"的形象。

而"有价值的人"——

在末世里，是最受尊重、也最安全的身份。


果然。

入城第五天，就有人来找她了。

不是普通人——是基地管理委员会的人。

来的是个中年男人，四十来岁，穿着干净的中山装，头发梳得一丝不苟，态度客气但带着审视。

"沈暮雪女士？"

"是我。"

"我是基地管理委员会的干事，姓周。我们主任想见您一面，方便吗？"

沈暮雪心里有数了。

管委会——基地的文职管理层。

他们找她，八成是因为听说了她的物资和空间系异能。

"方便。"她点点头，"什么时候？"

"现在就可以。"

沈暮雪跟姜糖交代了一声，然后跟着周干事往基地的核心区走去。

青北基地的布局分四个区：

A 区——核心区，管委会、军方指挥部、通讯中心、医疗中心都在这里。建筑最好，防御最强。

B 区——异能者居住区，觉醒了异能的人集中住在这里，待遇比普通区好。

C 区——普通居住区，大部分普通幸存者住在这里。

D 区——物资仓储和加工作坊区。

沈暮雪现在是 C 区的身份，但异能登记为空间系（稀有），所以走完流程后大概率会被调到 B 区。

而管委会找她——

可能就是谈这件事。

同时也想了解她的物资情况。

沈暮雪心里很清楚——

这是一场谈判。

她有基地想要的东西：物资，尤其是新鲜的蔬菜和药品。

基地有她想要的东西：安全的环境、合法的身份、更大的平台。

双方都有需求。

谈判的核心是——条件。

她能给出多少？

她能换回什么？

沈暮雪走在基地的路上，看着四周忙碌而有序的人群，心里越来越平静。

前世的她，只是末世底层的一粒尘埃。

这一世——

她站在了青北基地的大门口，即将走进那个决定无数人生死的核心圈。

她不再是一粒尘埃。

她是一颗正在升起的——

星。

深吸一口气。

"走吧。"她对周干事说，"带路。"
