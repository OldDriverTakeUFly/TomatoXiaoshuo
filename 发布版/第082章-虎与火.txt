第八十二章 虎与火

变异虎动了。

它没有正面冲——而是横向移动，沿着城墙根快速跑动，试图绕到侧翼。

"想绕后？"沈暮雪冷笑。

她举起弩——火箭上弦——

"嗖！"

金红色的火箭射向变异虎的移动路线前方——

不是射它——是射它即将经过的位置。

预判射击。

变异虎在跑到那个位置的瞬间——火箭在它脚边炸开——

"嘭！"

火焰在地面爆开，形成一道火墙——挡住了它的迂回路线。

变异虎被迫急停，转身——

沈暮雪的第二发火箭已经到了——

这次是直射。

正中变异虎的左肩——旧伤位置。

"嗷——！！"

变异虎怒吼一声——金红色的火焰在它的旧伤处燃烧，烧进了皮甲的裂纹里。

疼。

上次暗红色的火只能烧表面——这次金红色的火焰直接渗透进了裂纹，灼烧底下的肌肉组织。

变异虎一甩身体，用灵气把火焰震散——但烧伤已经造成了。

它的左肩——开始流血了。

暗金色的皮甲裂纹处，渗出了暗红色的血。

"第一刀。"沈暮雪低声说。

变异虎的琥珀色眼睛里——怒火翻涌。

它不再迂回了——

直接冲。

四米长的身体，以全速——朝城墙冲来。

"嘭——！！"

它撞上了城墙——

城墙震了一下，但没有塌——霍北的加固扛住了。

但变异虎没有退——它用前爪抓住城墙表面，开始往上爬——

C 级的力量——它的爪子像钢钩一样插进混凝土里，一步一步往上攀。

它要爬上城墙——

直接攻击沈暮雪。

"它上来了！"城墙上的士兵惊呼。

"退后！"沈暮雪喊——她不能让普通人卷入 C 级的战斗。

士兵们后退。

沈暮雪一个人——站在城墙边缘。

变异虎的头从城墙边缘冒了上来——

琥珀色的眼睛，獠牙，血盆大口——

近在咫尺。

沈暮雪没有退。

她双手合十——金红色的火焰在掌心凝聚到极致——

火焰领域——

开启。

以她为中心，十米范围内的温度骤升——

800℃。

空气扭曲了。城墙表面的混凝土开始发红——快被烤化了。

变异虎的前爪刚搭上城墙顶部——

它的爪子——被高温灼烧了。

"嗷——！"

它本能地缩回了爪子——C 级的皮甲能挡住火焰，但800度的持续高温，连皮甲都扛不住太久。

特别是旧伤位置——裂纹处的温度更高——火焰直接从裂纹钻进去，烧它的肌肉和神经。

变异虎从城墙上滑了下去——

摔在地上。

它踉跄着站起来，左肩的烧伤在扩大——暗金色的皮甲在高温下开始剥落。

受伤了。

真正意义上的受伤——不是上次的皮外伤。

沈暮雪站在城墙上，俯视着它。

"上次——你一掌拍碎了我的城墙。"

"这次——你的爪子连墙都爬不上来。"

变异虎的琥珀色眼睛里——

愤怒在消退。

取而代之的是——

犹豫。

C 级变异兽的智商足够高——它已经判断出来了：

这个人类——比它强。

不是强一点——是压倒性地强。

继续打下去——它可能会死。

变异虎后退了一步。

然后又一步。

"想跑？"沈暮雪的眼神冷了下来。

上次——它跑了。

这次——

她不会给它跑的机会。

它跑了之后会进化、会带更多的同伴来、会成为基地永远的隐患——

必须——

今天杀死。

沈暮雪从城墙上跳了下来。

直接跳到了城墙外面——

落在变异虎面前十米处。

"沈暮雪！你疯了？！"城墙上传来陈雪的惊呼。

沈暮雪没有回头。

她面对着变异虎——

金红色的火焰在她全身燃烧——

不是局部的火球、火蛇——

是全身覆盖的——

火焰铠甲。

C 级的新用法——把火焰领域压缩到体表，形成一层攻防一体的铠甲。

温度——800度。

任何接触到她的东西——都会被瞬间烧毁。

变异虎看着她——

一个浑身裹着金红色火焰的人类——

朝它走来。

它的本能——终于战胜了愤怒。

转身——跑。

C 级变异虎——以全速逃跑。

四米长的身体在荒野上飞奔——速度惊人。

但沈暮雪——

她没有追。

她站在原地，双手抬起——

把体内所有的 C 级灵气——

集中在一点。

炎爆——

不是对准变异虎——

是对准它逃跑路线前方十米处的地面。

"轰————！！！"

一团金红色的火球——从她掌心飞出——

落在变异虎前方的地面上——

爆炸。

方圆十米——毁灭性的高温。

地面被炸出了一个三米深的坑——坑壁的泥土和石头被高温融化成了琉璃。

爆炸的冲击波和火焰——从变异虎的正前方反扑过来——

变异虎无路可逃——

正面是爆炸的火墙——

后面是沈暮雪——

它被困在了——

火与火之间。

"嗷————！！！"

变异虎发出了一声最后的、绝望的嚎叫——

金红色的火焰从四面八方包围了它——

800度的温度——持续灼烧——

暗金色的皮甲在高温下开始龟裂、剥落、融化——

它的皮——挡不住了。

C 级的皮甲可以挡住 D 级的火焰——

但挡不住同等级的、持续性的、800度的领域级高温。

"嗷——……"

嚎叫声越来越弱。

变异虎的身体——在火焰中——

倒了下去。

沈暮雪站在原地——

维持着炎爆的余焰——

直到变异虎彻底不再动弹。

然后——

她收回火焰。

金红色的光芒消散。

荒野上——只剩下一个被烧成焦黑的、四米长的轮廓。

和一颗——

静静地躺在灰烬中的、拳头大小的、泛着暗金色微光的——

C 级晶核。

沈暮雪走过去，弯腰捡起那颗晶核。

入手——滚烫。

带着变异虎最后的体温。

"这次——"她低声说，"你没有跑掉。"

她转身，走向城墙。

城墙上传来——

震天的欢呼声。
