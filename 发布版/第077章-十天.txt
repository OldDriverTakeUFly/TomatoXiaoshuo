第七十七章 十天

倒计时开始。

沈暮雪给自己定了一个近乎不可能的目标——十天内突破 C 级。

从 D+ 到 C，差了一个大级别。

正常情况下，这需要几个月甚至半年的积累。

但她不正常。

她有空间、有灵泉、有灵果、有小豆子的寻宝天赋精准定位晶核——

还有裴夜。

"你的问题在于——灵气积累不够。"裴夜在训练场对她说，"D+ 到 C 的门槛，不是技巧的问题，是总量的问题。你需要大量的高等级晶核。"

"多大量？"

"至少——三颗 C 级晶核。或者——十五颗 D 级晶核。"

C 级晶核——只有在 C 级变异兽身上才有。

而 C 级变异兽——不是她想找就能找到的。

D 级晶核——她可以通过狩猎获取。

但十五颗——意味着她要在十天内猎杀至少十五只 D 级变异兽。

每天一点五只。

听起来不多——但 D 级变异兽不是路边的野草，不是想遇就能遇到的。

除非——

"我带你去找。"裴夜说。

"你知道哪里有 D 级？"

"基地西北方向三十公里，有一片变异兽聚居区。"裴夜说，"灵气浓度比基地附近高——吸引了很多 D 级个体在那里活动。"

"前世——我带队去过那里清剿。知道路线和兽群分布。"

沈暮雪看着他。

"你——亲自带我？"

"不亲自打。"裴夜说，"我带你找到目标，你打。我在旁边兜底。"

"如果遇到 C 级呢？"

"我来。"

沈暮雪想了想，点头。

"好。明天出发。"

"嗯。"

她转身准备回去收拾装备——

"沈暮雪。"

"嗯？"

"你的灵气总量——还有一个提升途径。"

"什么？"

裴夜看着她，沉默了一下。

"灵泉。"

"灵泉？灵泉水我一直在喝——"

"不是喝。"裴夜说，"是——泡。"

"泡？"

"你的空间里有灵泉眼。"裴夜说，"灵气最浓郁的地方——是泉眼本身。如果你能浸泡在灵泉眼中——身体会直接从泉眼吸收灵气。效率比喝高出十倍。"

沈暮雪愣了一下。

她之前确实没试过——因为灵泉眼很小，只有一个小水池那么大，而且她一直不确定直接浸泡是否安全。

"你试过吗？"她问。

"没有。"裴夜说，"但我前世见过类似的案例——有一个空间系异能者，通过在空间灵泉中浸泡修炼，半年内从 D 级突破到了 B 级。"

"半年从 D 到 B？"

"他的空间品质不如你。"裴夜说，"你的先天亲和度更高——速度应该更快。"

沈暮雪的心跳加速了。

灵泉浸泡——直接吸收灵气——效率十倍于喝——

如果这是真的——

十天突破 C 级——不是不可能。

"我今天晚上就试。"她说。


当晚。

沈暮雪回到板房，锁好门，拉上窗帘。

盘腿坐在床上，意识沉入空间。

灵泉眼。

那口一直汩汩涌出的泉水，汇聚成一个小水池——直径约两米，水深约一米。

泉水清澈见底，微微泛着蓝色的光芒。

沈暮雪走到水池边，脱掉鞋袜，把脚探了进去——

温热。

不是热水的温热，是一种从皮肤渗透进经脉的、带着灵气的温热。

她深吸一口气，慢慢坐进了水池里。

泉水没过她的腰。

然后是胸口。

然后——肩膀。

她整个人浸泡在了灵泉中。

效果——

立竿见影。

灵气从四面八方涌来——不是从嘴里进入的那种缓慢、温和的吸收——

而是从皮肤的每一个毛孔，直接灌入经脉。

像是一条干涸的河道，突然被洪水灌满。

沈暮雪能感觉到体内那个"火种"——在灵气的冲击下，急剧膨胀。

从拳头大小——变成了篮球大小。

从篮球大小——变成了磨盘大小。

它在燃烧。

不是燃烧灵气——是燃烧她体内的杂质。

D+ 级的瓶颈——那道看不见的壁障——在灵泉的冲击下，开始出现裂纹。

脑海里的提示音响了——

【检测到宿主处于高浓度灵气环境。】

【灵气吸收效率：提升 1000%。】

【火系经验值快速积累中——】

【当前等级：D+ 级。】

【突破进度：12%……23%……37%……】

沈暮雪闭上眼，全力引导灵气运转。

她的身体在灵泉中微微发光——暗红色的火系灵气和蓝色的灵泉灵气交织在一起，形成了一道绚丽的、螺旋状的光流。

时间在空间内流速是外界的十倍——

她在空间里泡了一个小时，外界只过了六分钟。

一个小时后——

【突破进度：58%……】

还不够。

但已经比她预想的快太多了。

如果在外面正常修炼——58% 的进度至少需要一个月。

而在灵泉里——一个小时。

沈暮雪退出灵泉，大口喘气。

她的身体——像是被洗过了一遍。

每一块肌肉都充满了力量，每一条经脉都畅通无阻，连皮肤都变得更加细腻光滑。

灵泉浸泡——不仅提升灵气，还净化身体。

"每天泡一个小时。"她在心里定下计划，"十天——足够突破。"

退出空间。

睁开眼。

窗外，月亮已经升到了中天。

沈暮雪躺在床上，看着天花板，嘴角弯了弯。

C 级。

她来了。


接下来的十天，沈暮雪的生活变成了两点一线——

白天：和裴夜出城狩猎，猎杀 D 级变异兽，回收晶核。

晚上：在灵泉中浸泡修炼，冲击 C 级瓶颈。

每一天，她都能感觉到自己变强了一分。

D+ 级的瓶颈在灵泉和晶核的双重冲击下——

一天天松动。

第五天——突破进度 72%。

第七天——85%。

第九天——94%。

第十天——

倒计时最后一天。

沈暮雪在清晨走进了空间。

灵泉。

她坐进去。

闭上眼。

引导所有灵气——冲击最后那道壁障。

94%……96%……98%……99%——

"嘭！"

她听见了一声——

像是玻璃碎裂的声音。

体内的壁障——

碎了。

脑海里的提示音响起——

【恭喜宿主！火系异能突破！】

【火系等级：D+ 级 → C 级。】

【新增能力：炎爆（C 级范围毁灭技）——将大量火系灵气压缩至一点引爆，造成方圆十米内的毁灭性高温伤害。】

【新增能力：火焰领域（完整版）——自身周围十米范围内，温度可自由控制（最高 800℃），对范围内所有目标造成持续灼烧。】

【提示：C 级火系已进入基地顶级战力梯队。】

C 级。

突破了。

沈暮雪从灵泉中站起来，看着自己的双手——

掌心里，一团比之前任何时候都更亮、更烈、更纯粹的火焰，在安静地燃烧。

颜色——

从暗红色，变成了金红色。

金红色的火焰。

C 级。

她做到了。

十天。

从 D+ 到 C。

沈暮雪握紧拳头，火焰在指缝间跳跃。

她抬起头——

空间的天花板上没有天空。

但她知道——

在外面，一轮新的兽潮，正在向青北基地逼近。

而她——

终于有了正面迎战它的资格。
