# 《末世重生：空间女皇》第二卷补充设定

> 本文档补充第二卷新登场的角色、势力和设定，与人物卡/世界观设定集配合使用。

---

## 一、新角色

### 孙建国（孙派系头目·第二卷主要反派）

| 项 | 内容 |
|----|------|
| 年龄 | 45 岁 |
| 外貌 | 方脸、板寸头、体格壮实、嘴角永远向下撇；穿军装但不系最上面扣子，透着一股随意 |
| 身份 | 青北基地军方副指挥官，上校军衔，韩志远的副手——但两人面和心不和 |
| 异能 | C 级力量强化型（物理系）——力量是常人二十倍，徒手能掀翻一辆车 |
| 背景 | 末世前是地方武警支队的副队长，末世后凭借武力和手腕迅速在基地里拉起了一支"私兵"，约三十个异能者效忠于他 |
| 性格 | 野心大、手段狠、善于伪装。在韩志远面前毕恭毕敬，背后搞小动作。信奉"末世里枪杆子就是一切" |
| 与陆衍关系 | 陆衍投靠他，充当"谋士"角色，提供沈暮雪的情报 |
| 目标 | 篡夺青北基地控制权，建立自己的军阀势力 |
| 下场 | 第二卷末被韩志远借兽潮内鬼事件清洗，削权降职。但人没死——留到第三卷继续作乱 |

**写作要点**：孙建国不能写成无脑莽夫。他有脑子，有手腕，能拉拢人心。他的失败不是因为蠢，而是因为格局不够——他只看到眼前的权力，看不到沈暮雪和裴夜背后更大的棋局。

---

### 南域使者·方明远（第三卷反派的前哨）

| 项 | 内容 |
|----|------|
| 年龄 | 38 岁 |
| 外貌 | 文人长相、戴金丝眼镜、温文尔雅、说话慢条斯理，笑起来让人如沐春风——但眼底没有温度 |
| 身份 | 南域基地"外务使"，负责与其他基地的外交（实为情报刺探） |
| 异能 | B 级精神系（读心·低级版——能感知对方情绪波动和是否说谎，不能读取具体想法） |
| 性格 | 阴柔、耐心极好、擅长话术。从不正面冲突，专背后捅刀 |
| 与陆衍关系 | 利用陆衍获取青北情报，用完即弃 |
| 作用 | 第二卷末出现，为第三卷"青北vs南域"主线埋钩子 |

**写作要点**：方明远是"笑面虎"型反派。他越客气，读者越应该警惕。他的精神系异能让他极难被欺骗——但沈暮雪的空间系和裴夜的精神系（隐藏）恰好克制他。

---

### 陈雪（基地女战士·非雌竞女配·凌霜的替代定位）

| 项 | 内容 |
|----|------|
| 年龄 | 25 岁 |
| 外貌 | 短发、英气、身材精瘦但肌肉线条分明；左眉有一道浅疤 |
| 身份 | 基地正规战斗队队长，B 级风系异能者 |
| 性格 | 直爽、讲义气、看不惯弯弯绕绕。对沈暮雪从"审视"到"敬佩"到"惺惺相惜" |
| 与沈暮雪关系 | 起初因为沈暮雪被编入战斗序列而不服（觉得她是靠物资上位的），后来在兽潮中亲眼见证她的实力，转为真心敬佩 |
| 作用 | 打破"女配都恶毒"套路。她是沈暮雪在基地里的女性战友，后期成为重要盟友 |
| 口头禅 | "少废话，打一架就知道了。" |

**写作要点**：陈雪和沈暮雪的关系线要写出"从不打不相识到并肩作战"的成长感。她不是附庸，是独立的强者。

## 二、势力格局更新

### 2.1 青北基地内部派系

```
                    青北基地
                   ╱        ╲
          韩志远（正统派）   孙建国（实权派）
          ╱    |    ╲         ╱    |    ╲
       赵德明  正规军  异能者  私兵   陆衍   依附者
       (管委会)(驻军) (战斗队)(30人) (谋士) (各种)
                   |
              沈暮雪团队（中立·独立）
              裴夜（超然·不受任何派系调度）
```

**派系矛盾核心**：
- 韩志远代表"秩序和集体利益"——他想把基地建设成真正的幸存者庇护所
- 孙建国代表"个人野心和丛林法则"——他想把基地变成自己的军阀王国
- 沈暮雪夹在中间，两边都想拉拢她，但她选择独立——这反而让两边都忌惮她

### 2.2 青北 vs 南域

| 对比 | 青北基地 | 南域基地 |
|------|----------|----------|
| 性质 | 军方主导+异能者自治，相对公正 | 强人独裁，资源垄断 |
| 人口 | 约 5 万 | 约 8 万 |
| 核心战力 | 裴夜（S级）、沈暮雪（B级·卷末）、陈雪（B级） | 疑似有 S 级强者（第三卷揭示） |
| 目标 | 守护幸存者 | 扩张吞并 |
| 与女主关系 | 主场 | 敌对 |

## 四、第二卷感情线节奏

```
Ch66 虐点：误解裴夜在调查她 → 冷战
     ↓
Ch67 转折：真相大白，裴夜是在保护她
     ↓
Ch71 和好：天台谈心，解开心结
     ↓
Ch72 告白：沈暮雪主动，"我信你" / 裴夜回应，"我守你"
     ↓
Ch73 甜宠日常：送饭、训练、吃醋
     ↓
Ch88 并肩作战：CP合击C级变异兽
     ↓
Ch102 卷尾：城墙看日出，"我守你"的延续
```

**感情线核心**：先虐后甜，虐是为了让甜更甜。虐点不能太长（2章），甜点要密集（确认关系后每5-8章一个糖）。

---

## 五、第二卷虐渣节奏

```
Ch64-65 陆衍造谣 → Ch69-70 第一次清算（失势）
Ch84    陆衍偷物资记录 → 被抓（作死）
Ch92    内鬼事件牵连 → 第二次失势
Ch94    彻底投靠南域 → 出卖情报
Ch97    当众清算 → 身败名裂
Ch98    异能被废 → 终极惩罚
Ch99    放逐 → 比死更狠
```

**虐渣核心**：吊着打，每次清算都让他跌得更深，但不到最后一步不杀——因为"让他活着看自己一无所有"比死更狠。

## 六、第二卷必守红线

1. **火系公开后不能无敌**——B级很强但不是最强，C级以上的威胁仍然存在
2. **裴夜不代劳**——C级变异兽必须沈暮雪参与击杀，裴夜是辅助不是主角
3. **虐渣不手软但要合理**——陆衍每次阴谋都有逻辑，不是送人头
4. **CP虐点不长**——冷战最多2章，超过就影响追读率
5. **孙建国不死**——他留到第三卷，第二卷只削权
6. **治愈系姜糖必须保密**——到第二卷末只有团队内部知道
